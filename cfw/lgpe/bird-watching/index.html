LGPE Overworld Legendary Bird Hunting (CFW)

This guide will cover the process of how you can find members of the legendary bird trio Articuno, Zapdos & Moltres in the overworld easily by using the moarencounterbots branch of the Sysbot.NET repository by Lusamine.

This guide assumes you are using a CFW Switch console running Atmosphere.

Getting Started:

You will need:
  • Sysbot.NET - Moarencouterbots - Latest Commit
  • sys-botbase - Latest release
Visit the official Sysbot.NET Wiki for instructions on building the application using Visual Studio

Installation Instructions:
  1. Download and extract the contents from the sys-botbase download into an empty folder.
  2. Place the atmosphere folder(s) from the sys-botbase download onto the root of your Switch SD card.
  3. Check the /atmosphere/contents directory of your SD card a folder named 430000000000000B.
  4. Reboot your Switch console and launch CFW.

Bot Setup:

In order to use the LGPE encounterbots, open the exe once and close it to generate a config.json. Right click this file and open it in a text editor and change the Mode to 100.

Figure 1 - Hub tab overview


The Hub tab is where decide the type of encounter we are doing, and which Pokemon to hunt for. Scroll down to EncounterLGPE and set your conditions, click on each of the fields and you will be met with a short description about them at the bottom of the window.

Now scroll down to stopconditions and set your filters for your legendary bird of choice, you can be as specific with the filters as you like, such as filtering for specific IVs or shiny type, or have the bot continue the routine if the legendary bird is not shiny.

Suggested settings and a detailed overview of each setting can be found on the , it's recommended to read this page in its entirety before starting, it contains a lot of useful information!

Figure 2 - Log tab overview


The Log tab is where we monitor what is going on in-game, it shows us what the bot is doing in real-time and will contain all the information you need to know, such as when a legendary bird has spawned in the overworld, or if it's a shiny variant.

Figure 3 - Bots tab overview


The Bots tab is where we add bots, select the EncBotBirdWatch option in the dropdown menu, type your switch IP into the field and click Add to add a new instance of EncBotBirdWatch.

With setup out of the way, it's time hunt some legendary birds!
