Item Printer RNG Guide

This guide provides in-depth coverage of how to manipulate the Item Printer machine in Scarlet & Violet. This is used for obtaining desirable items for capturing and training Pokemon, such as rare Pokeballs, Ability Patches, Vitamins etc.

Requirements
In-game Requirements
  • Access to the DLC expansion pass on the game you are using.
  • Item Printer machine unlocked in the Blueberry Academy Clubroom.
  • Access to Stellar Tera Shards from capturing Terapagos.
  • Item Printer machine fully upgraded to Master Ball Rank.
  • Plenty of Pokemon Materials in order to use the machine.
See Serebii's Item Printer page for more information on how to fully upgrade the Item Printer.
Program Requirements
  • ItemPrinterDeGatcha
  • Epoch Time Converter
  • Any timer (seconds)
ItemPrinterDeGatcha only runs natively on Windows OS. If you are using a different OS and cannot run this application it may be more difficult to calibrate second-precise timing.

Once you have everything set up properly, stand next to the Item Printer machine in-game, then save.

Getting Started

All of the items you receive from the Item Printer are predetermined based on the consoles date & time, which can be manually altered in the Switch System Settings (Home screen -> Settings -> System -> Date and Time).

Item Printer Mechanics
Seeding

As mentioned above, the items you get from the item printer are based on the system date & time. By understanding which times generate your desired items, you can easily control the output/result.

The machine has second-precise timing, so a timer is required as the Switch does not have a native method of viewing the amount of seconds that have passed in real time. The timer can be installed on your PC, on a mobile phone, or any external device.

The seed you hit is determined when the screen fades to black after selecting the I want to print something! option. The amount of time it takes to fade to black can vary based on your console (typically 1-2 seconds), this requires calibration.

Bonuses

Bonuses are randomly given to a player after using the machine, these essentially grant bonus rewards and extra print jobs (aka uses) of the machine. These can also be manipulated to control the end result.

There are two different types of bonuses that can be applied, these are:

  1. Item Bonus - Uses the item table to determine rewards
  2. Ball Bonus - Uses the ball table to determine rewards

These bonuses can apply a minumum of 10 extra print jobs, taking advantage of print jobs is the most effective way to farm items.

Delay Calibration

The term "delay" is used go describe the amount of time it takes the screen to fade to black. Delay needs to calibrated and taken into consideration in order to hit desired seeds/results.

In the above image, we can see that the screen fades to black in 1 second, which means that or calibrated delay should be 1 second every time unless we change setup (different console, etc).

Step-by-Step
  • Interact with the Item Printer, then hover over the I want to print something! option.
  • Open your external timer, set it to calibrate the amount of seconds that pass if necessary.
  • Start the timer, then click the A button to start a print job.
  • Take note of the amount of seconds that pass until the screen fades to black.
If you are having difficulties troubleshooting your delay, a more accurate method using ItemPrinterDeGatcha is described below.

Figure 2 - Seed Finding



Seed Finding

If you are having difficulty finding your delay and are using a machine running Windows OS, the ItemPrinterDeGatcha can be used to help you find exactly which seed you are hitting.

Open ItemPrinterDeGatcha and head over to the Adjacent tab. This tab is used for troubleshooting purposes.

Step-by-Step
  • Follow the delay calculation steps mentioned above (here)
  • This time, take note of the exact time the screen fades to black (date, hour, minute & second).
  • Also take note of the item that was produced by the machine.
  • Convert the exact time into a seed using the Epoch Time Converter.
  • Paste the seed into the Seed field of ItemPrinterDeGatcha's Adjacent tab.
  • Use the +1 or -1 buttons to find the item(s) you actually hit.
  • Calculate how far this was from your original seed in seconds, this will be your new delay.
In most cases your delay should be 1-2 seconds. If you are not finding any matching results in the Adjacent tab, make sure that the seed is correct by double checking the date & time.
