RaidCrawler Guide (CFW)

This guide will provide coverage of how scan Tera Raid dens in Scarlet/Violet by using the RaidCrawler program developed by LegoFigure11. RaidCrawler can be used to read all of the Tera Raid dens on the map, this can be used to scan for shininess and/or specific IV values.

This guide assumes you are using a CFW Switch console running Atmosphere. This is also technically NOT RNG abuse! This is considered closer to brute force RNG than true RNG abuse.

Getting Started:

You will need:
  • RaidCrawler - Latest release
  • sys-botbase - Latest release
  • .NET 6.0 - Program dependency
Installation Instructions (Click to expand!)
  1. Download and extract all the contents from the RaidCrawler.zip into an empty folder.
  2. Download and extract all the contents from the sys-botbase download then place the atmosphere folder on to the root of your Switch SD card.
  3. Check the /atmosphere/contents directory of your SD card for a folder named 430000000000000B.
  4. Reboot your Switch console and launch CFW.

Tool Setup:

Figure 1 - RaidCrawler UI Overview


RaidCrawler UI Overview
Pictured above is RaidCrawler, by connecting our console to this program we can scan all of the active Tera Raids on the map, and even continue until a shiny Pokemon is found using automated date skipping!

To start with, enter the local IP address of your Switch console into the Switch IP: field, then click connect. You should now notice that the status has changed to Reading raid block..., then set your game version and story progression level inside the dropdown menus on the left.

Game progression key (Click to expand!):
  1. New game
  2. 5 gym badges obtained
  3. 8 gym badges obtained
  4. Elite 4 beaten
  5. 6 star raids unlocked

  • The Read Raids button can be used at any time, this is particularly useful if you are manually advancing the date.
  • The Advance Date button can be used to automate the date skipping process, be sure to desync any controllers before clicking this button or it will not work.
  • When the Continue until shiny? checkbox is active, the program will advance the date until a shiny Tera Raid appears, this is extremely useful for players who don't want to date skip and check manually.
  • The arrow buttons are used to check the contents of each raid seperately. Each raid is indicated with a unique number /69.
  • When the Event? checkbox is active, this means that the Tera Raid den currently being read is a promotional distribution Tera Raid.

Note: In order to use the Date Skip feature it is advised to desync all controllers currently connected to your Switch console or have only Joy-Cons attached to the console.

You should also disable Syncronize Clock via Internet in your console system settings.

Note: Remove any overlays, cheats or cheating applications such as Edizon, Tesla and Capturesight if you are experiencing read errors.


Figure 2 - Shiny Found!


After scanning the raid dens and advancing the system date using Day Skip feature, we were able to quickly find a shiny raid and we have been provided with the following information:

  • Seed, PID and EC, the PID will be highlighted yellow if it is a shiny Pokemon
  • Pokemon species and Tera typing
  • The general location of the raid on the map
  • Raid difficulty
  • The IVs of the raid (For accurate reads it is important to set your game progression level correctly)
  • If raid is an event distribution
By using this information we can navigate towards the area that contains our shiny Pokemon to confirm the result in-game matches the output on RaidCrawler.
