BDSP PokeRadar Chain RNG (CFW)

This section of the guide covers how to RNG PokeRadar chain length, this type of RNG can be used to reach IV/Shiny bonuses with the PokeRadar. Chain length RNG abuse currently requires specific tools and raw scripts, so some additional download & setup instructions are required.

It is still assumed that you have followed all of the setup & install instructions found on the main page of this guide, as some of these tools will be used here, despite the need for additional downloads.

PokeRadar Setup Instructions:
You will need:
  • LINQPad
  • PokeRadar Script & PKHeX.Core.dll
Installation Instructions:
  1. Download LINQPad, then run the installer and go through all the setup instructions.
  2. Download scripts.zip and extract the contents into the same folder as LINQPad.

LINQPad Script Setup:

Upon opening the script in LINQPad, you should see something similar to the above image. The script should work fine with no modifications, if you are running into issues, do the following:

Leveraging PKHeX.Core.dll
  • With the script open, press F4 to bring up the Query Properties sub-window.
  • In the Additional References tab, click the Add/Browse button
  • Select PKHeX.Core.dll, this should be found inside the dlls file created earlier.
  • Open the Namespace Imports tab then type PKHeX.Core.
  • Click the OK button at the bottom left of the sub-window to confirm these changes.
  • Run the script, you should see that an output has been produced at the bottom of the main window.
Ensure that the script is being ran as a C# Program, this is visible in the top bar of the main window. Use Figure 1 as a reference point if you are struggling to leverage PKHeX.Core.dll.

Finding Current Seed

To easily obtain your current seed at any given point you can use the EncBotCopySeedBS bot routine in Moarencounterbots. You will need to toggle a few settings in the Hub tab for use with PokeFinder.

CopySeed Setup:
  • In the Hub tab, open the EncounterRNGBS dropdown menu.
  • Select Bit128 in the DisplaySeedMode dropdown menu.
  • Set the MonitorRefreshRate to 0, this will be needed later.
  • In the Bots tab, type the local IP address of the console.
  • In the dropdown menu on the right, select EncBotCopySeedBS and add the bot.
  • Click the Start All button, your seed will be produced inside the Logs tab.

These seeds can be pasted into the string ram = "0123456789ABCDEF0123456789ABCDEF"; field of the script to search for frames that will not break your chain (paste your seed between the two "").

It is not recommended to change any other aspects of the script code unless you know what you are doing, this will result in inaccuracies going forward.

PokeRadar Rules

First, you will need to understand the "rules" of the PokeRadar, Serebii's BDSP PokeRadar page contains a lot of useful information about chain continuation and shiny odds.

The script assumes you have the highest odds of keeping the chain alive, achieved by targeting tiles on the most outer ring and capturing the Pokemon (93%), by targeting Pokemon in the outer-most ring and capturing them.

Guaranteed Chain Breaks
  • Encountering a random wild Pokemon
  • Running from a radar encounter
  • Mounting your bike
  • Leaving the area
Increase Chain Continue Odds
  • Encounter the Pokemon 4 tiles away
  • Capture the Pokemon instead of defeating it
  • Keep a repel active at all times
Tips & Recommendations
  • Bring enough repels and disable autosave in case of a failed chain
  • Bring a high leveled Breloom (or alternative) to easily capture Pokemon
  • Take the bike off the list of registered key items to avoid using it
Your capture Pokemon should be a higher level than the wild Pokemon in the area to utilize the repel trick, but it should also not be level 100. The script also relies on your Pokemon being able to gain exp after a battle.
