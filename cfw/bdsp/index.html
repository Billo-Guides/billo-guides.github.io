BDSP RNG (CFW)

This guide provides in-depth coverage of how to RNG abuse encounters, gifts and eggs in Brilliant Diamond & Shining Pearl using tools & scripts developed by Lusamine & AdmiralFish. Using RNG abuse you can obtain Shiny, high IV Pokemon with desirable natures, as well as calculating other values such as gender and size.

From here it is assumed you are using a CFW Switch console running Atmosphere, this guide is also geared towards users running Windows OS machines.

Setup Instructions:
You will need:
  • PokeFinder
  • SysBot.NET - Moarencounterbots
  • sys-botbase
Microsoft C++ Redistributable is required to run Pokefinder on Windows machines. .NET 8.0 SDK is required to run .NET based applications. Ensure both of these dependencies are properly installed before continuing.
Installation Instructions:
  1. Download PokeFinder.zip, then extract the contents into a known location on your computer.
  2. Download the SysBot.NET - Moarencounterbots source code, then extract the contents into a known location on your computer.
  3. Download sys-botbase.zip and extract the atmosphere folder to the root of your Switch SD card. Reboot the console into SysCFW.
Upon rebooting the console, a blue light should appear around the Home button of your Player 1 controller to indicate that sys-botbase has been properly installed. If you do not see a blue light you will need to start the installation process again.

It is recommended to build the latest commit of Moarencounterbots if you are able to do so. A pre-compiled download is included above to assist those who are unable to build the program themselves. If you wish to compile the application yourself, click the button directly below.
