BDSP RNG (CFW)

This guide provides in-depth coverage of how to RNG abuse encounters, gifts and eggs in Brilliant Diamond & Shining Pearl using tools & scripts developed by Lusamine & AdmiralFish. Using RNG abuse you can obtain Shiny, high IV Pokemon with desirable natures, as well as calculating other values such as gender and size.

From here it is assumed you are using a CFW Switch console running Atmosphere, this guide is also geared towards users running Windows OS machines.

Setup Instructions:
You will need:
  • PokeFinder
  • SysBot.NET - Moarencounterbots
  • PokeViewer.NET
  • Sys-botbase
Microsoft C++ Redistributable is required to run Pokefinder on Windows machines. .NET 8.0 SDK is required to run .NET based applications. Ensure both of these dependencies are properly installed before continuing.
Installation Instructions:
  1. Download PokeFinder.zip, then extract the contents into a known location on your computer.
  2. Download the SysBot.NET - Moarencounterbots source code, then extract the contents into a known location on your computer.
  3. Download PokeViewer.zip, then extract the contents into a known location on your computer.
  4. Download sys-botbase.zip and extract the atmosphere folder to the root of your Switch SD card. Reboot the console into SysCFW.
Upon rebooting the console, a blue light should appear around the Home button of your Player 1 controller to indicate that sys-botbase has been properly installed. If you do not see a blue light you will need to start the installation process again.

It is recommended to build the latest commit of Moarencounterbots if you are able to do so. A pre-compiled download is included above to assist those who are unable to build the program themselves. If you wish to compile the application yourself, click the button directly below.

Getting Started

From ths point onwards, it is assumed that you are able to run all of the above mentioned applications, and that sys-botbase has been properly installed.

It is important to understand how to use each of the applications used for the RNG abuse process. If you are unfamiliar with how to use PokeFinder, Moarencounterbots or PokeViewer, this is the place to start.

Figure 1 - PokeFinder Overview



PokeFinder Overview:

PokeFinder is used for predicting upcoming advancements based on a given RNG seed. Launching PokeFinder for the first time you will be presented with a small window, this is where you will select the game you wish to RNG abuse in, in this case you will be searching for Brilliant Diamond & Shining Pearl found under the Generation 8 tab. After selecting it, a larger "main window" will appear.

Pictured above is the main window, this is where all of the relevant information about the Pokemon you plan to RNG abuse will be entered, this window is the centralized hub used for most types of RNG abuse in Brilliant Diamond & Shining Pearl.

  • The left hand side is for search conditions, here we can select the type of encounter, enter our seeds, and input the amount of advances to search through.
  • The central portion is where specific details about the Pokemon will be entered, such as species, and parents if the chosen type of encounter is an egg.
  • The right hand side is for inputting search filters, this is used for finding specific desirable results, such as only outputting shiny or high IV frames.
  • The top portion is for profile management, input your trainer information such as OT, TID, SID, Game & Variables. Your 5 digit TID/SID is only required for Egg RNG abuse.
Finding your TID/SID

It is recommended to use PKHeX in order to find your 5 digit TID/SID combination. You will need the latest version of PKHeX and a way to extract your save file to do this. Tutorials can be found online.

  • Open your save file in PKHeX with at least 1 self obtained Pokemon in your party or boxes.
  • View any Pokemon by right clicking it, then clicking the View button.
  • Navigate to the OT/Misc tab, here you will see the trainer details of the Pokemon.
  • Hover over the TID field using your mouse, you will see two 5 digit values (e.g 12345(TID)/54321(SID).
  • Take note of these values and enter them into the profiles sub-window of PokeFinder, then save and close the window.

Ensure you have the correct profile loaded for the save file you are currently using! Especially if you intend to RNG abuse eggs. Eggs are the only encounter type that have a shiny odds increase from the Shiny Charm.

Figure 2 - Moarencounterbots Overview



Moarencounterbots Overview:

Pictured above is the window you are presented with upon launching Moarencounterbots - SysBot.exe.. Moarencounterbots is used for monitoring RNG seeds and automating the main bulk of the RNG advancement process.

To connect your console to SysBot.NET you will need the local IP address, this can be found by opening the Homebrew Menu on your console, then pressing the Y Button. Enter this IP address into the assigned field on SysBot. Bot type can then be selected using the drop-down menu on the right.

There are four bot types that will be used throughout this guide, these are:

  1. EncBotRNGMonitorBS - Used to monitor your RNG seed in real time.
  2. EncBotCopySeedBS - Used to copy your RNG seed to clipboard at any point.
  3. EncBotTIDBS - Used to automate the process of TID RNG abuse.
  4. EncBotDexFlipBS - Used to automatically advance the RNG state until a given point.
General information and usage tips can be found on the official Moarencounterbots Wiki. We recommend becoming familiar with each of these bot types before continuing.

Figure 3 - PokeViewer.NET Overview



PokeViewer Overview:

Pictured above is the window you are presented with upon launching PokeViewer.exe. PokeViewer is used to read the details of a Pokemon during a battle or found inside boxes, which is necessary for troubleshooting after missing a target.

To connect your console to PokeViewer.NET you will need the local IP address, this can be found by opening the Homebrew Menu on your console, then pressing the Y Button. Enter this IP address into the assigned field on PokeViewer, then connect after launching the game.

There are two main features used throughout this guide, these are:

  1. EncounterView - Used to view the details of an opposing Pokemon during a battle.
  2. BoxView - Used to view the details of a Pokemon located inside boxes.
Usage tips can be found on the official PokeViewer Wiki. We recommend becoming familiar with PokeViewer usage before continuing.

You should now have a general understanding of how to use the necessary tools, it's time to get to work!
