BDSP Troubleshooting

Information about troubleshooting encounters or redemptions.

Missing your target and not sure how to fix the problem? This section covers everything you need to know about troubleshooting any type of encounter or redemption. We will be using a combination of PokeFinder and PokeViewer.NET, if you have made it this far both of these applications should already be installed on your computer.

Depending on which encounter or redemption type you are troubleshooting the process may differ slightly, for this reason the information for each encounter/redemption type has been branched off into it's own section. Select the appropriate encounter/redemption type in the dropdown menu below to get started.
