BDSP Trainer ID RNG (CFW)

This section of the guide covers how to RNG manipulate your Trainer ID for cosmetic purposes. This type of RNG abuse is not required for manipulating specific spreads or Shiny Pokemon and can be skipped if you are not interested in specific TIDs.

During gameplay, BDSP has a constantly advancing seed that determines the outcome of random events, such as what Pokemon you will encounter, the stats it will have, if the encounter is shiny, etc. The same value also controls which Trainer ID you will receive upon starting a new save file.

From here it is assumed that you have read and followed all of the setup and installation instructions found on the initial page of this guide.

TID RNG Setup:
What is TID RNG?

This type of RNG abuse allows you to RNG abuse your visual (6 digit) TID to have a save file that has the exact trainer ID value you desire. As stated above, this is purely for cosmetic purposes.

Moarencounterbots is capable of automating 95% of the process by using the bot routine EncBotTIDBS. You will need to configure the bot properly in order to search for desirable target frames first.

Settings Configuration

Settings can be found by opening the Hub tab, here we can toggle the # of frames to search through and create a list of desired Trainer IDs to stop on by doing the following:

  • In the EncounterRNGBS dropdown menu, navigate to MaxTIDAdvances, then set the amount of advances to search through (10000 advances takes roughly 2.5 hours).

  • In the same dropdown (EncounterRNGBS), set the MonitorRefreshRate to 0, this will be important for the 5% of work we need to do ourselves.

  • In the StopConditions dropdown menu, navigate to TargetTIDBS, then enter a comma seperated list of acceptable TIDs (e.g 000001, 010101, 111111, etc.).

Once you have configured your settings, it is good practice to restart the bot entirely to lock them in place. After doing so, add the EncBotTIDBS bot by doing the following:

  1. Type the Local IP address of your console into the assigned field (how to find your Local IP is covered on the main page).

  2. Select the EncBotTIDBS routine from the dropdown menu on the right of the SysBot window.

  3. Click the Add button, you should now see the bot has been added to the list on the Bots tab.

  4. Open the game with no save file loaded, you should be on the language select screen at this stage, now disconnect any controllers and click Start All.

  5. The bot will now begin cycling through frames and resetting the game until a result has been found, then advance to roughly ~15 frames from the target and enter the home menu once complete.
The bot assumes that your desired language is first on the selection screen, change your Switch language setting to your desired language if you have not already done so.

Hitting A Desired Result:
Monitoring the RNG state:

Moarencounterbots is capable of monitoring the Main RNG state in real time and able to track the amount of advances that have occured since monitoring began. This is done using the EncBotRNGMonitorBS routine:

  1. Confirm that MonitorRefreshRate is set to 0, as described above.

  2. With the console in the in the HOME menu, navigate to the Bots tab of Moarencounterbots, then add EncBotRNGMonitorBS.

  3. Click the Start All button to run the bot, you should now see that the RNG state is being monitored in the Logs tab.

  4. Watch the logs until the correct amount of advancements have passed, then quickly press the A button, your TID will be generated at this point.

  5. Name your character and mash through all the dialogue until you are standing in the overworld in-game, then reconnect the bot. You should see your OT and Trainer ID in the logs.
Example - If there are 12 advancements until we reach the target frame, wait until 12 advancements have passed according to the logs before pressing the A button.
