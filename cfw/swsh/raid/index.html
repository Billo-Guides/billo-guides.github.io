Max Raid RNG Guide (CFW)

This guide provides in-depth coverage of how to use PokeFinder by AdmiralFish to RNG abuse Max Raid Battles in Sword/Shield. Using raid RNG abuse you can predict the internal values of upcoming raids (IVs, Nature, Shiny Status etc.) and filter for desirable results.

From here it is assumed you are using a CFW Switch console running Atmosphere.

Don't have CFW installed? Click here for the retail focused guide!
Setup Instructions:
You will need:
  • PokeFinder
  • Visual C++ Redistributable (Windows OS)
  • Luxray Date-skipper
Installation Instructions:
  1. Download the latest version of PokeFinder, make sure to download the version specific to the OS you are using, then extract it into a clean, empty folder.
  2. For Windows machines, download Visual C++ Redistributable and click the installer file, go through all of the setup instructions.
  3. Download Luxray.zip, and extract the contents of the folder into an empty folder in a known location on your computer.
  4. Move the atmosphere and switch folders from the Luxray download to the root of your Switch SD card, then reboot the console into SysCFW.
