Max Raid RNG Guide (CFW)

This guide provides in-depth coverage of how to use PokeFinder by AdmiralFish to RNG abuse Max Raid Battles in Sword/Shield. Using raid RNG abuse you can predict the internal values of upcoming raids (IVs, Nature, Shiny Status etc.) and filter for desirable results.

From here it is assumed you are using a CFW Switch console running Atmosphere.

Don't have CFW installed? Click here for the retail focused guide!
Setup Instructions:

The requirements will differ depending on the type of Operating System you are running, while the rest of this guide will be focused on Windows OS, other platforms can still follow along using their own set of tools. Open the dropdown menu specific to the operating system you are using and follow all of the setup & installation instructions.

WINDOWS OS

You will need:
  • PokeFinder
  • PKHeX + LiveHex
  • sys-botbase
  • Luxray Date-skipper
Dependencies are required to run programs, download Visual C++ Redistributable & .NET 8.0 SDK, then run the installer before continuing, most people will require the x64 (64 bit) versions.
Installation Instructions:
  1. Download the latest version of PokeFinder for Windows, then extract it into a clean, empty folder.
  2. Download the latest version of the PKHeX Plugins installer (setup_stable.exe), then extract it into a clean, empty folder.
  3. setup_stable.exe, after setup is complete you will see PKHeX.exe and a plugins folder in the same directory.
  4. Download and extract the contents of the sys-botbase download into an empty folder inside a known location on your computer. Then download Luxray.zip and extract the contents into the SAME folder.
  5. You should now two folders named atmosphere and switch, move these folders to the root of your Switch SD card, then reboot the console into SysCFW.

Confirm that everything has been set up properly before continuing.

  • PokeFinder should launch and run properly with no issues or pop ups.
  • PKHeX should launch and run properly with no issues, and the taskbar icon should be BLUE to indicate that plugins were installed correctly.
  • Sys-botbase should make the Home button of your controller glow BLUE when installed to the console correctly.
  • Luxray should be present in the Homebrew menu and should launch and function with no issues.
If you are running into issues installing PKHeX or Sys-botbase, visit the relevant Wiki pages found on the GitHub repository.
MAC / LINUX OS

You will need:
  • PokeFinder
  • Tesla Menu
  • NXOvloader

  • Luxray Overlay
