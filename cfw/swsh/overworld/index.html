Overworld RNG Guide (CFW)

This guide will provide in-depth coverage of how to RNG abuse all types of overworld-based encounters in Sword/Shield by using the SWSH Overworld RNG Generator by LegoFigure11. Overworld RNG abuse can be used to obtain shiny and/or marked Pokémon from wild encounters! This can also be used to calculate other internal values such as IVs, nature, and gender.

This guide assumes you are using a CFW Switch console running Atmosphere.

Click here for the retail RNG abuse guide (unmodified Switch console)!

Getting Started:

You will need:
  • SWSH Overworld RNG Generator - Latest release
  • .NET 6.0 - Generator dependency
  • sys-usb-botbase - Latest release
  • ldn-mitm - Latest release
Installation Instructions (Click to expand!)
  1. Download and extract all the contents from the SWSH_OWRNG_Generator_GUI.zip into an empty folder.
  2. Download and extract all the contents from the sys-usb-botbase & ldn-mitm downloads into an empty folder.
  3. Place the atmosphere folder(s) from these downloads onto the root of your Switch SD card.
  4. Check the /atmosphere/contents directory of your SD card for two folders named 430000000000000B & 4200000000000010.
  5. Reboot your Switch console and launch CFW.

Main Window Overview
Pictured above is the Main Window of the generator. This window is used to calculate upcoming target frames based on our search parameters.

To start with, enter the local IP address of your Switch console into the Switch IP: field, then click connect. You should now notice your current seeds, and TID/SID combination have been automatically filled into the correct fields.

  • Init. adv. is the amount of frames that have passed since connecting to the generator. The + field below is the amount of frames to search through. Set this to an amount you are comfortable with.

  • The checkboxes on the left-hand side should be filled out according to your desired target. We can quickly find this information using the Encounter Lookup tab.

  • The Shiny Charm & Mark Charm checkboxes should be checked if those items are present your save file. It's recommended to aquire both of these items before starting.

  • The Read Encounter will read any wild encounter mid-battle. This can be used to quickly troubleshoot delays. (More on this later!)

Figure 2 - Encounter Lookup Overview


Encounter Lookup Overview
This is the Encounter Lookup window. Searching for a specific species in the dropdown list will bring up a list of every possible wild encounter for that Pokémon in the entire game.

How to use the Encounter Lookup:
  1. Open the Encounter Lookup window and set the game version to the game you are currently playing.

  2. Type the species name of your chosen target Pokémon, then select it from the dropdown menu. You should now see a list of all possible encounters for that specific Pokémon in the table.

  3. Choose an encounter type; Static, Symbol or Hidden. Once you have decided, select one of the rows from the list by single clicking on it.

  4. The details of the encounter should now be automatically filled into the main window, and the Encounter Lookup window can now be closed.


Figure 3 - Menu Close Advances Viewer Overview


Menu Close Prediction Overview
The Menu Close Advances Viewer can be used to predict the amount of advancements that will pass from closing the pause menu. Closing the pause menu also has an added effect of freezing the RNG state in the area for a a few seconds, this makes it a great option for closing in on any target frames as it provides a lot more consistency in doing so.

By knowing how many advancements will occur from closing the pause menu, and having a window of silence to run into position and spawn the encounter, we can near guarantee to hit any potential target frames.

How to calculate the number of NPCs in an area:
  1. Press the X button on your controller to open the pause menu, then disconnect and reconnect to the main window. This will ensure the number inside the Curr. Adv remains at 0.

  2. Click the Menu Close Advances Viewer button at the top of the main window to open a smaller window. Your seeds should be visible in the correct fields.

  3. Close the pause menu then quickly re-open it, take note of the number of advancements that have occured from doing so.

  4. Enter your number of advancements into the Init Adv. field. If the above steps were closely followed, this number should be 0.

  5. Type 0 into the NPCs field and search. Gradually increase this number by 1 until you find a match. Repeat this step a few times to ensure the advancement predictions are accurate.



Advancing the RNG State

The RNG state will advance naturally depending on your location, the weather of the location, and the spawned Pokémon/NPCs that are present. In all cases, it's best to target a Pokémon in a "quiet" area with calm weather conditions. However, sometimes you will be forced into an unstable environment due to a lack of options.

Examples:
 - In Crown Tundra, the area is naturally quiet, so we will have a relatively easy time hunting a Pokémon in clear weather conditions.
- In Isle of Armor, the area is naturally noisy, so we will have a more difficult time hunting Pokémon in general.
- If we added rainy weather to either of these situations, we would increase the amount of natural noise and further increase the instability.

Automating Your Advancements:
Day Skipping
The Skip field of the main window is used for day skipping. Simply type the number of days to skip into the empty field, then click the Days button. (If this does not work, enter the Switch system settings and sync the system clock via internet).

The amount of advancements from day skipping is heavily influenced by the amount of Pokémon currently stored within boxes. Fill your boxes with Pokémon to make the most of this feature. (This does not include Pokémon in eggs!)

Avoid date skipping in large distances inside any of the Wild Areas (including Isle of Armor and Crown Tundra Wild Areas). This may cause your game to crash due to the constantly advancing Max Raids! We recommend using an in-door location to advance distances >1000 days.

Motions/Animations
The Adv. button can also be used in a similar way, party animation advancements only advance 1 frame at a time, this can be used to accurately move closer towards the target frame.
