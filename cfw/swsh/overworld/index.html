Overworld RNG Guide (CFW)

This guide provides in-depth coverage about how to RNG abuse overworld-based (Wild) encounters in Sword/Shield, using the SWSH OWRNG Generator GUI created by LegoFigure11. Overworld RNG abuse can be used to obtain Shiny and/or Marked Pokémon from wild encounters! This can also be used to calculate other values such as IVs, Nature, & Gender.

From here it is assumed you are using a CFW Switch console running Atmosphere.

Don't have CFW installed? Click here for the retail focused guide!
Setup Instructions:
You will need:
  • SWSH Overworld RNG Generator
  • ldn-mitm
  • Luxray Date-skipper
Installation Instructions:
  1. Extract SWSH_OWRNG_Generator_GUI.zip into an empty folder on your PC in a known location.
  2. Extract the contents of both ldn-mitm.zip & Luxray.zip into the same folder as the above.
  3. You should now see an atmosphere & switch folder inside this folder. Drag these folders to the root of your Switch SD card.
  4. If done correctly your SD card should look similar to this image. Plug the SD card back into the console and reboot SysCFW.
Note - .NET 6.0 is a requirement to use the SWSH OWRNG Generator GUI. Ensure that this has been properly installed before continuing.

RNG Tool Setup

This section will cover the essential details you need to know in order to use the Generator. If you do not yet have a complete understanding of how each of the tabs, follow along with this section of the guide as it will provide a relevant information & usage tips.

Figure 1 - Main window



Main Window Overview

Pictured above is the main window, this window is used to calculate upcoming target frames based on your search parameters. Here we can connect the Switch console to the program and automatically track our current RNG seed.

Type the local IP address of your Switch console into the Switch IP: field, then open the game. Once you are inside the overworld clicking the Connect button, your current seeds and TID/SID combination will be filled into the correct field automatically. Do not tweak these values after connecting to the generator.

  • Init. adv. displays the amount of advances that have passed since connecting to the generator.
  • The + field is the amount of frames to search through, set this to an amount you are comfortable with.
  • Shiny Charm & Mark Charm checkboxes should be checked if you have these items unlocked, otherwise leave these unchecked.
  • The middle section contains search filters, here you can set filters for specific Shininess, IVs, Nature, Marks, etc.
  • Read Encounter is used to read the details of wild Pokemon while in-battle, useful for troubleshooting delays.
  • Skip is used to automate the process of advancing frames using a CFW device, using the Days and Adv. buttons.
  • The rest of the boxes on the left-hand side can be filled out automatically using the Encounter Lookup tab.

Figure 2 - Encounter Lookup Overview



Encounter Lookup Overview

Pictured above is the Encounter Lookup tab, here we can search for specific Pokémon species names to generate a list of every possible encounter for that specific Pokémon in the game.

How to use Encounter Lookup:
  1. Open the Encounter Lookup tab, then set the correct game version according to the game you are currently playing.
  2. Type the species name of your chosen target Pokémon, then select it from the dropdown menu to see a list of all the possible encounters.
  3. Choose an encounter type; Static, Symbol or Hidden. Once you have decided, select one of the rows from the list by single clicking on it.
  4. The details of the encounter should now be automatically filled into the main window, Encounter Lookup can now be closed.

Figure 3 - Menu Close Advances Viewer Overview



Menu Close Advances Viewer Overview

Pictured above is the Menu Close Advances Viewer tab, here we can predict the amount of advancements that will occur from closing the pause menu. After entering your seeds and hitting search a table will appear, as seen in the image above.

The Jump column will display the amount of advancements that will occur from closing the pause menu, this number will vary depending on the amount of NPCs that are in the area that you plan to RNG abuse in.

An area can have a large number of NPCs even if you can not visually see any, so it is important to properly calibrate your NPC count depending on the area you are in so that an accurate Jump is produced by the Menu Close Advances Viewer tab.

Menu close prediction is recommended for closing in on target frames, closing the pause menu will freeze the RNG state for a few seconds, which make it an extremely useful method that will significantly increase your chances of hitting the correct target frame in most cases.

How to effectively calculate NPC count will be covered at a later stage of the guide.


Advancing The RNG State

The RNG state will advance naturally depending on your location, the weather of the location, and the spawned Pokémon/NPCs that are present. The best case scenario is to target a Pokemon in an area where the RNG state naturally advances at a slow rate, however, sometimes you will be forced into unstable conditions depending on the Pokémon you are hunting.

Avoid rain/thunderstorm weather where possible! These weather types cause the RNG state to advance in a rapid & inconsistent manner that will make hitting a desired target frame much more difficult. Any other weather should be easy enough to work with.

Date Skipping Information

It is possible to automate the date-skipping process using CFW. This is extremely useful for covering large amounts of advancements while idle. Regardless of the method used, time and date must be synced via the internet in the Switch system settings.

As stated in the table, the amount of frames that advance from day skipping is largely variable, depending on the amount of Pokémon currently stored inside box slots. Fill your boxes with Pokémon (NOT EGGS) to make the most of this feature. Take note of the number of frame advancements that occur per day skipped to avoid overshooting any target frames.

Avoid date skipping in large distances inside any of the Wild Areas! This may cause your game to crash due to the constantly advancing Max Raids and changing weather conditions. We recommend using an in-door location to advance distances >1000 days.

Skipping using Luxray (recommended):

Luxray is a calculator-style application that can be launched through the Switch Homebrew Menu. It has a variety of features, such as:

  1. Set - Sets the network time to a desired date, based on the figure typed into the calculator. Useful for changing the weather.
  2. Step - Initiates date skipping, based on the figure amount typed into the calculator. Used to automate date-skipping.
  3. NTP - Sets the date and time back to network time, this should be done after every date-skipping session.

Skipping using OWRNG tool:

To skip using SWSH_OWRNG_Generator_GUI, type the number into the Skip, then click the Days button. You will need to wait for the countdown to finish before attempting to day-skip again.

This method does not include an NTP function, you will also need to set the correct weather condition after finishing the date skipping process. This can be done safely by changing the date backwards until the correct weather conditions have been met. Changing the date backwards will not cause unwanted advancements.
