Abilities & Dex Recommendations

This section explains advanced techniques involving the use of lead abilities and the using Pokedex recommendation feature. These methods are used to help determine:

  • The details of a spawned Pokemon (e.g., nature and gender).
  • The probability of spawning a particular Pokemon in the encounter table.
  • Increasing the likelihood of triggering a hidden encounter.

This table provides information about each of the lead abilities and their use cases:

Ability Overworld Effect
Synchronize The nature it has will apply to any encountered overworld Pokemon upon encountering, this can be used after the Pokemon has been spawned.
Cute Charm If your lead Pokemon is the same species as the spawned Pokemon, increase the chances of that Pokemon having the opposite gender.
Magnet Pull If there are STEEL TYPE Pokemon in the encounter table, increase the chances of spawning those Pokemon, otherwise, do nothing.
Lightning Rod & Static If there are ELECTRIC TYPE Pokemon in the encounter table, increase the chances of spawning those Pokemon, otherwise, do nothing.
Flash Fire If there are FIRE TYPE Pokemon in the encounter table, increase the chances of spawning those Pokemon, otherwise, do nothing.
Storm Drain If there are WATER TYPE Pokemon in the encounter table, increase the chances of spawning those Pokemon, otherwise, do nothing.
Harvest If there are GRASS TYPE Pokemon in the encounter table, increase the chances of spawning those Pokemon, otherwise, do nothing.
Compound Eyes & Super Luck If the Pokemon can be spawned holding a random held item, increase the likelihood of this occuring, otherwise, do nothing.
Arena Trap, Illuminate & No Guard INCREASE the chances of spawning a hidden encounter, these abilities are important to use when manipulating hidden encounters.
Quick Feet, Stench, White Smoke & Infiltrator DECREASE the chances of spawning a hidden encounter, in most cases, these abilities should not be used on your lead Pokemon.

Pokedex Recommendations

Upon opening the Pokedex, four Pokemon icons will be present on the left side of the screen, these four Pokemon are granted an increased spawn rate until the date is changed. This feature is commonly referred to as Pokedex recommendations or Dexrec. If the date has changed since the last time the Pokedex was opened, or if the Pokedex has not been opened at all, it is considered inactive.

Figure 1 - Dexrec Example

In the above example, NidoranF, Sinistea, Heatmor and Magby are the currently boosted Pokemon, ANY encounter table that contains these Pokemon will have this boost applied! This means that a particular Dexrec is global and not restricted to a specific area.

Only the first form ID of a particular Pokemon will be boosted, for example, if our current Pokedex reccomendation is Sinistea, only the phony form will be boosted as this is considered to be Sinisteas first form, it is not possible to boost the antique form.

Anubis has provided a research thread about dex recommendation for those wanting to learn more.

Note - Currently, we do not have any way to meaningfully control which Pokemon is boosted by Dexrec, for this reason, and combined with the fact that dex recommendations are cleared when using the date skipping advancement method, it will be difficult to utilize this method.

If you are not using date skipping, ensure that your dex recommendation has been set before you begin advancing towards a target, otherwise, leave the dex recommendation inactive.
