Automation Methods

Learn how to use all of the CFW automation features provided by the RNG tool.

This page provides a demonstration on how to use the automation features provided by owoow. Here you can learn how to effectively date skip, automate animation advances, and reset for a desirable seed. Automation/CFW only features can be found on the far right of the owoow window.

Figure 1 - Date Skipping Automation

To automate the date skipping process, simply stand in the overworld, then type the amount of days you wish to skip into the Skip: field (see image) and click the Days+ button. If this was done correctly you should notice that the RNG state has begun advancing rapidly. After the date skipping has completed, you can press the NTP button (Network Time Protocol) to reset your current system time & date according to the current network time.

Important - Avoid date skipping over 1000 days inside any of the Wild Areas, as this is likely to cause the game to crash. To avoid this, simply date skip large amounts inside an in-door location, such as a PokeCenter.

Figure 2 - Animations Advancing Automation

For animation automation, open the pause menu and enter the Pokemon party list, then type the chosen amount of advances into the Skip: field (see image) and click the Adv. button. You should now see that the game has begun automating animation advances. As a reminder, each animation will cause a +1 advance to the current RNG state (100 animations = 100 advances).

Important - As this method requires automated button pressing, your JoyCons must be attached to the console. Alternatively (if you are not using JoyCons), you can disconnect all controllers after entering the party summary screen instead.

Figure 3 - Seed Resetting

Clicking the Seed Reset Settings option will open a sub-window, here is where you can set conditions for owoow to automate the seed hunting process.
