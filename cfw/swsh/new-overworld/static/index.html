Overworld Static Encounters

Worked examples for manipulating Static encounters in any weather condition.

Static spawners consistently produce the same species until the weather in the area changes, which can occur at midnight (00:00). Be mindful of the current system time to avoid spawning unexpected Pokemon due to incorrect weather conditions.

Important - It is assumed that you are familiar with the basic concepts outlined on the main page of this guide. If you are unsure about topics such as seed monitoring, advancement methods, or NPC calibration, please revisit the relevant pages before starting.
Figure 1 - RNG Tool Setup

Getting Started:
  1. Select Encounter Lookup from the toolbar at the top of the main window of owoow to open the sub-window.

  2. Select a game version on the right, then type the species name of your desired Pokemon and select an encounter.

  3. Take note of the encounter type, location and weather condition, then enter this information into the Encounter Settings column of the main window.

  4. Set your search range (the + field in Encounter Settings), select a lead ability, then set your search filters. You can filter for Shiny, Mark, IVs, Height, etc.

  5. Fly to the correct location in-game, then locate the spawner that contains your desired Pokemon. After doing so, run far enough away to despawn it and save the game.

  6. Type your IP address into owoow, then click the Connect button while standing in the overworld (ideally with the game paused), then begin searching for target frames.
Undesirable search result?

Increase your search range. Alternatively, restart the game and disconnect/reconnect to owoow, this will give you a fresh initial seed. You can also use owoow's Reset For Seed feature to automate the process of searching for a desirable seed.

Figure 2 - Game Setup

Game Setup:

If you have not already done so, locate the spawner that contains your desired Pokemon in game, stand far away enough to completely despawn it, then save the game. You can tell if you are far enough away from the spawner by walking into it and looking for a dust cloud at the foot of the Pokemon. If you do not see a dust cloud, you are standing too close.

The above location in Giant's Seat is ideal: no additional spawned Pokemon, no active NPCs, and Giant's Seat is notably "quiet" during calm weather conditions. A "quiet" area refers to an environment where the RNG naturally advances at a slow pace.

If you cannot find a good position, you can knock out any extra Static Pokemon blocking your path. These will remain despawned until the date is changed, the area is refreshed by entering a loading zone, or the game is rebooted after a date advance.


With all of the required initial setup out of the way, let's get to work!
