Overworld Static Encounters

Worked examples for manipulating Static encounters in any weather condition.

Static spawners consistently produce the same species until the weather in the area changes, which can occur at midnight (00:00). Be mindful of the current system time to avoid spawning unexpected Pokemon due to incorrect weather conditions.

Important - It is assumed that you are familiar with the basic concepts outlined on the main page of this guide. If you are unsure about topics such as seed monitoring, advancement methods, or NPC calibration, please revisit the relevant pages before starting.
Figure 1 - RNG Tool Setup

Getting Started (Tool Setup):
  1. Type your IP address into owoow, then click the Connect button while standing in the overworld (ideally with the game paused).

  2. Select Encounter Lookup from the toolbar at the top of the main window of owoow to open the sub-window.

  3. Enter the name of your desired species into the assigned field, then select a STATIC encounter. Then take note of the location and required weather condition for your chosen encounter.

  4. Close the Encounter Lookup window, then enter the location, weather and species into the Encounter Settings column of the main window.

  5. Set your search range (the + field in Encounter Settings), then set your search filters. You can filter for Shiny, Mark, IVs, Height, etc.

At this point, it will be assumed that you are successfully connected to the program and have your desired target/location/weather condition in mind. You do not need to search for a target frame yet.


Figure 2 - Game Setup

Game Setup:
  1. Fly to the area that contains your desired Pokemon, set the required weather condition in-game, then run around until you find the correct spawner.

  2. Run far enough away to completely despawn the Pokemon, then save the game. You can tell if you are positioned correctly by looking for a dust cloud at the foot of the Pokemon when spawning it in the overworld.

  3. If you cannot find a good position, you can knock out any additional static spawners that are blocking your path, those will remain despawned until the date is advanced and the area is reloaded.

  4. Once you are positioned correctly you will need to set your lead Pokemon if you plan to use one, then save the game once more. If you are unsure which lead Pokemon to use click the button directly below.

The above location in Giant's Seat is ideal: no additional spawned Pokemon, no active NPCs, and Giant's Seat is notably "quiet" during calm weather conditions. A "quiet" area refers to an environment where the RNG naturally advances at a slow pace.
