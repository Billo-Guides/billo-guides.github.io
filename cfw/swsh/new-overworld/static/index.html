Overworld Static Encounters

Worked examples for manipulating Static encounters in any weather condition.

Static spawners consistently produce the same species until the weather in the area changes, which can occur at midnight (00:00). Be mindful of the current system time to avoid spawning unexpected Pokemon due to incorrect weather conditions.

Important - It is assumed that you are familiar with the basic concepts outlined on the main page of this guide. If you are unsure about topics such as seed monitoring, advancement methods, or NPC calibration, please revisit the relevant pages before starting.
Figure 1 - RNG Tool Setup

Getting Started (Tool Setup):
  1. In the main window, type your consoles IP address then click Connect, this will automatically read your current seed and the details of the save file.

  2. Select Encounter Lookup from the toolbar at the top of the owoow main window to open the sub-window, then type the name of your desired species.

  3. Search for a STATIC encounter, then enter the Location/Weather/Species information into the Encounter Settings section of the main window.

  4. Adjust your search range (the + field in Encounter Settings), then configure your search filters such as filtering for Shiny, Mark, IVs, or Height.

At this point, it will be assumed that you are successfully connected to the program and have your desired target/location/weather condition in mind. You do not need to search for a target frame yet.


Figure 2 - Game Setup

Game Setup:
  1. Fly to the area that contains your chosen encounter, set the required weather condition in-game, then run around until you find the spawner that corrisponds to your desired Pokemon.

  2. Run far enough away to completely despawn the Pokemon, then save the game. You can tell if you are positioned correctly by looking for a dust cloud or puddle at the foot of the Pokemon when spawning it in the overworld.

  3. If you cannot find a good position, you can knock out any additional static spawners that are blocking your path. Those will remain despawned as long as you do not move far away or reload the area.

  4. Once you are positioned correctly you will need to set your lead Pokemon if you plan to use one, then save the game once more. Detailed information about lead abilities and their use cases can be found by clicking the button below.

The above location in Giant's Seat is ideal: no additional spawned Pokemon, no active NPCs, and Giant's Seat is notably "quiet" during calm weather conditions. A "quiet" area refers to an environment where the RNG naturally advances at a slow pace.
