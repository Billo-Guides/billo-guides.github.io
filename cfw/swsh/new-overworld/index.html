Overworld RNG Guide (CFW)

This guide provides in-depth coverage of how to use the Sword/Shield Overworld RNG tool (owoow), a tool created and developed by LegoFigure11.

Introduction

Overworld RNG manipulation is used to predict the internal details of upcoming wild encounters. It is possible to predict values such as IVs, Nature, Marks, Shiny Status, Size (Jumbo/Teensy Mark), Gender and more. This table contains information about the types of encounters that can be manipulated, provides example images, and outlines the benefits of picking each encounter type.

Encounter Description Difficulty
Static

Encounters that are guaranteed to produce a specific species, provided weather conditions remain unchanged.

These encounters are not eligible for Brilliant Aura, meaning they cannot benefit from boosted shiny odds or IV bonuses. 		Easy
Symbol

Random encounters that occur in grass, water, and caves, with species determined by an encounter slot value and current weather conditions.

These encounters are eligible for Brilliant Aura, allowing for boosted shiny odds and IV bonuses. Players aiming for high IV targets may wish to prioritize this encounter type whenever possible. 		Easy
Hidden

Random encounters that occur in grass that do not have an overworld model. These encounters are indicated by the "!" symbol when walking into grass.

These encounters are not eligible for Brilliant Aura, meaning they cannot benefit from boosted shiny odds or IV bonuses. 		Advanced
Fishing

Random encounters in fishing pools found in water, with species determined by an encounter slot value and current weather conditions.

These encounters are eligible for Brilliant Aura, allowing for boosted shiny odds and IV bonuses. Players aiming for high IV targets may wish to prioritize this encounter type whenever possible. 		Intermediate

Setup Requirements

This section covers everything you need in order to get started. Read these instructions carefully to avoid complications and potential troubleshooting.

You Will Need:
Tools & Applications:
  • .NET 9.0
  • Sword/Shield Overworld RNG Tool (owoow)
  • ldn-mitm
Installation Instructions:
  1. Download .NET 9.0 and run the setup wizard until complete.
  2. Download and extract owoow.zip into an fresh, empty folder.
  3. Download and extract ldn-mitm.zip into the same folder as the above.
  4. Move the atmosphere and switch folders to the root of your Switch SD card.
  5. Reboot the console into your chosen CFW environment (SysCFW recommended).
Program Connectivity Instructions:
  1. Launch owoow.exe, ensure that the program launches properly with no issues.
  2. Type the local IP address of the console into the Switch IP: field.
  3. Enter the overworld in-game, then click Connect on owoow after doing so.
  4. If done correctly, the program will now begin reading the details of your save file, and the seed fields on the top left will now be populated.

Important - If you are experiencing connectivity issues with the program, please ensure that both sys-botbase (bundled with owoow) and ldn-mitm are correctly installed on your SD card. If the issue persists, consider removing any additional software that could be interfering with the gameâ€™s RAM, such as cheats or mods.

RNG Tool Overview

This section provides program information and usage tips, if you do not already understand each of the features of the RNG tool, this is the place to be.

Figure 2 - Owoow Overview

RNG Tool Overview:

Upon launching and connecting to owoow for the first time, you should see something similar to the above image. This tool calculates the details of upcoming Pokemon, then outputs the results in the table at the bottom (based on user-defined search parameters).

Feature Overview:
  • Left Side - Provides seed information, connectivity features, and save file details such as Trainer IDs and Game Version.
  • Center - Select encounter details, calibrate NPC count, enter search range, and input other specific configurations.
  • Right Side - Input search filters, automate RNG state advancements, and read encounter details and more.
Automated Monitoring:

After connecting to the program, the details of your save file (including IDs, Shiny/Mark Charm, and Game Version) will be automatically populated. Adjustments to these values are not required. Additionally, RNG seeds will be monitored in real-time, providing a seamless experience for tracking seed changes.

The Adv: field displays the total number of advancements since the initial connection in the first box, and the number of advancements occurring within the last ~500ms in the second box. The second box is primarily used to determine how many advancements a specific action consumes.

Menu Close Prediction:

Menu close prediction is a technique used to advance the RNG state in a predictable manner in order to near guarantee success in hitting a specific target frame. This method is used extensively throughout the guide, regardless of the type of RNG you plan to do. It is highly recommended to understand how to utilize menu close prediction before continuing.

Automated Advancements & Seed Resetting:

It is possible to automate certain methods used to advance the RNG state, such as date skipping and animations. It can also be used to automate the process of resetting for a desirable seed, this cuts out a significant portion of manual effort. It is recommended to read through the information in this page if you are interested in owoow's automation features.
