Overworld RNG Guide (CFW)

This guide provides in-depth coverage of how to use the Sword/Shield RNG tool, developed by LegoFigure11. Overworld RNG abuse can be used to manipulate the internal details of wild encounters, such as the IVs, Nature, Gender, Size, Marks, Shiny status and more.

From here it is assumed you are using a CFW Switch console running Atmosphere.

Don't have CFW installed? Click here for the retail focused guide!
Setup Instructions:
You will need:
  • Sword/Shield RNG Tool (owoow)
  • .NET 8.0
  • ldn-mitm
Installation Instructions:
  1. Download and extract owoow.zip into an empty folder in a known location on your computer.
  2. Run the .NET 8.0 installer and go through the instructions given by the setup wizard, then confirm the RNG tool launches properly.
  3. Download and extract ldn-mitm.zip into the same folder from earlier, if done correctly the atmosphere folder will merge automatically.
  4. Open the atmosphere folder and confirm that there are two numbered folders inside atmosphere/contents/ (sys-botbase and ldn-mitm).
  5. Move the atmosphere folder to the root of your Switch SD card, then reboot the console into SysCFW.
Program Connectivity Instructions:
  1. Launch owoow.exe, then type the local IP address of the console into the Switch IP: field.
  2. Head to the overworld in-game, then click the Connect button, if done correctly the program will begin reading the details of your save file.
  3. If the seed fields (located on the upper left) have been populated, setup has been done correctly and you can continue following the guide.
If you are having issues when connecting to the program, ensure that sys-botbase and ldn-mitm have been properly installed, the console has been restarted and is using SysCFW, and that your console and PC are connected to the same network.

Tool Setup & Usage

This section will cover the essential details you need to know in order to use the RNG tool effectively. If you do not yet have a complete understanding of how each of the tabs, follow along with this section for relevant information & usage tips.

RNG Tool Overview:

Upon launching the RNG tool (owoow) for the first time, you should see something similar to the above image. This tool calculates the details of upcoming Pokemon, then outputs the results in the table at the bottom, based on your search parameters.

Feature Overview:
  • Left Side - Provides seed information, connectivity features, and save file details.
  • Center - Select encounter details, calibrate NPC count, and enter search range.
  • Right Side - Input search criteria, automate RNG state advancements, and read encounter details.
RNG Seed Monitoring:

This program allows real-time monitoring of your current RNG seed, automatically tracking the number of advancements that have occured since the console was connected. This creates a seamless experience for tracking seed changes.

In the Adv: field, the first box shows the total advancements since connecting, while the second box displays advancements over the last ~500ms, which is useful for tracking how many advancements are consumed from performing specific in-game actions.

Upon connecting to the program, your 5 digit trainer IDs and Shiny/Mark Charm will be auto-filled, you do not need to tweak these values.

Encounter Details:

You can use the Encounter Lookup button to open a sub-window, here, you can search for all the encounters in the entire game for each specific species. Simply type the name of your desired Pokemon and scroll through the table.

Before selecting an encounter, it is important to understand the differences of each of the four encounter types. Information and examples for each encounter type can be found in the table below.

After evaluating the table, pick an encounter that matches your skill level, then take note of the location and weather condition it can be found in. Avoid encounters in rain or thunderstorm where possible as these weather conditions create a lot of instability.

Advancing The RNG State

The RNG state will naturally advance based on your location, the weather, and the presence of spawned Pokémon & NPCs. Ideally, you should target a Pokémon in an area where the RNG state advances slowly. However, in some cases you may be forced into unstable conditions depending on the type of encounter you are hunting.

In-game actions can also be used to advance the RNG state with varying influence, the most useful actions can be found in the table below.

Advancement Automation:

The program is capable of automating RNG advancements to an extent, date skipping can be automated for large distance advancements, and animation advancements can be automated for advancing a single frame at a time.

Date Skipping:

Date-skipping is the fastest way to advance the RNG state in Sword/Shield, it is possible to advance upwards of 15,000 frames per day skipped. The amount of frames skipped by date skipping depends on the amount of Pokemon currently stored in box slots (not including eggs).

To automate the date-skipping process, simply type the amount of days you wish to skip into the empty field on the far right, then click the Skip button when standing in the overworld in-game. If done correctly you should notice that the RNG state has started to advance rapidly. After completing date skipping, you can use the NTP button to re-sync the system date according to the internet.

Take note of the amount of advancements that occur per day skipped, this will give you a rough idea of how many days need to be skipped in order to be close enough to your target frame. Advancements per day / target frame = amount of days to skip.

Avoid standing in any of the Wild Areas when automating date-skipping as this may cause the game to crash. It is recommended to date skip inside an in-door location, like a PokeCenter.
Animations:

When closing in on a target frame, you may wish to automate animation advancements instead of doing these manually. The process to do this is the same as date-skipping, simply type the number of animations you wish to automate, then press the Adv. button when on the party summary screen. If done correctly you should notice the RNG state advancing one-by-one automatically.

If you are using joy-cons, ensure that they are attached to the console before attempting to automate animation advancements, if you are using the pro controller, ensure that it is fully disconnected. This is a requirement to using this feature.

Picking An Encounter Type

You should now have a basic understanding of how to use the RNG tool and each of it's features. Use the encounter lookup feature to pick your chosen encounter type, then click the button that applies to you to get started.
