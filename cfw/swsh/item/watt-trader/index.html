Watt Trader RNG (CFW)

A guide for manipulating the daily highlight item from the Watt Trader NPC.

The Watt Trader NPC is located at the bottom of Snowslide Slope in the Crown Tundra. Each day, this NPC has a different highlighted item, some of which are the rare Dream/Beast Balls. This guide will focus on manipulating his daily highlight item for a desirable result.

Important - From here, we assume that you are familiar with key concepts described on the main page. If you are unsure about seed monitoring, advancement methods, or NPC calibration, please review the relevant sections before continuing.
In-game Requirements
  • Access to the Crown Tundra DLC.
  • Plenty of Watts to spend on the highlight item (each rare Pokéball costs around 40,000 Watts).
  • Interacted with the NPC at least once to skip the introduction and unlock the correct fly point.

Getting Started (Tool Setup)
  1. Enter the IP address of your console in the main window and click Connect. This will automatically read your current seed and save file details.

  2. Set your current weather condition inside the Encounter Settings section of the main window, this will make the process slightly more simple.

    • Avoid using Rain/Thunderstorm, these weather conditions are unsupported as they cause unnecessary instability.

  3. At the top of the main owoow window, select Watt Trader to open the Watt Trader generator in a subwindow.

  4. Enter your search filters, then enter the amount of advances you are willing to search through into the + field, usually you will find a result that produces a Dream/Beast Ball in less than 10,000 advances.

    • Dream/Beast Balls are the rarest items, if you are targeting something else you can expect to see these items in fewer advances on average.

  5. Set the correct weather condition if you have not already done so. This should be set inside the Watt Trader sub window automatically if you followed step 2.

  6. Select the Consider Menu Close? checkbox, but do not calibrate your NPC count yet as we require a specific in-game setup (explained below).

Figure 2 - Game Setup

Getting Started (Game Setup)
  1. Head to the location pictured above, then interact with the Watt Trader NPC at least once to unlock to the fly point if you do not already have access to it.

  2. Check your weather condition by opening the map. If the current weather is Rain or Thunderstorm be sure to change it as these conditions cause instability.

  3. Now you will need to position yourself optimally, this is done by using a movement technique that disrupts the NPCs "active" state and can achieved by doing the following:

    • Run towards the NPC and interact with them in one quick, smooth motion, then back out of the NPC dialogue.

    • If done correctly, you should see the advances occuring at a slow rate after backing out of the dialogue.

    • If done incorrectly, the NPC will activate after backing out of the dialogue, causing the usual, fast, NPC activation advances.

    • It may take a few attempts to get the NPC into the correct state, but this provides the most stable experience possible for this type of RNG manipulation.

  4. Calibrate your NPC count as normal, the expected NPC count when standing next to the NPC in Snowslide Slope is 4. Enter your calibrated NPC count into the Watt Trader sub window on owoow.

  5. Perform a date advance to refresh the daily highlight item, this can be done using owoow's date skipping features or the PvP date skipping exploit.

  6. Save the game. This will preserve the current NPC state upon resetting. Do not speak to the NPC again until you reach your target frame.

Advancing Towards a Target

In this example, a frame that produces a Beast Ball can be seen on advance 100, this is the exact frame that we will aim to close the pause menu on and then interact with the NPC to lock in our desired result.

The first two days of daily highlights will be locked in after speaking to the Watt Trader NPC, which means our target item will be available on the third day. To get to our desired highlight item we will need to perform 2 date skips and interact with the Watt Trader twice after we have done the RNG.

Step-by-Step
  1. As we need to maintain our current positioning, advance towards the target using any method that does not cause the player to move.

    • Consider using wet weather or date skipping to advance large distances (for targets > 10,000 advances).

    • Consider using animations or menu close prediction to advance shorter distances (for targets < 10,000 advances).

  2. Stop advancing once you are ~1000 advances before your target, then click the Update Seeds button on the main window. Now close and re-open the Watt Trader sub window.

  3. Ensure that the Consider Menu Close? checkbox is checked in the sub window, confirm that your NPC count and weather have been entered correctly, then search for your target again.

  4. From this point, use animations to reach the exact target frame, then return to the main pause menu.

  5. Unpause the game and interact with the Watt Trader NPC to lock in your desired item. To access your desired highlight item follow the steps directly below.
Accessing Our Daily Highlight
  1. Exit out of the Watt Trader dialogue.

  2. Perform a date skip.

  3. Interact with the Watt Trader, putting us on "day 2".

  4. Exit out of the Watt Trader dialogue.

  5. Perform another date skip

  6. Interact with the Watt Trader again, putting us on "day 3".

  7. Check Today's Highlight, this should be your desired item.

In Figure 4 we closed the pause menu on the desired target frame and quickly interacted with the Watt Trader. Now we need to select Today's Highlight to lock in our expected reward, then date skip using the above method to have him produce our desired item.

After date skipping and interacting with the Watt Trader enough times using the method described above, we can see that Today's Highlight features out desired item!
