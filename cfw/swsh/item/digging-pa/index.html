Digging Pa RNG (CFW)

A guide for manipulating the Digging Pa to farm a large number of Watts.

The Digging Pa NPC is located on the north side of the Isle of Armor. Interacting with him gives you the option to trade Armorite Ore in return for attempts at his minigame, where you can earn a large amount of Watts.

Important - From here, we assume that you are familiar with key concepts described on the main page. If you are unsure about seed monitoring, advancement methods, or NPC calibration, please review the relevant sections before continuing.
In-game Requirements
  • Access to the Isle of Armor DLC.
  • Armorite Ore to give to the NPC.
  • Interacted with the NPC at least once to skip the introduction.

Figure 1 - RNG Tool Setup

Getting Started (Tool Setup)
  1. Enter the IP address of your console in the main window and click Connect. This will automatically read your current seed and save file details.

  2. Set your current weather condition inside the Encounter Settings section of the main window, this will make the process slightly more simple.

    • Avoid using Rain/Thunderstorm, these weather conditions are unsupported as they cause unnecessary instability.

  3. At the top of the main owoow window, select Digging Pa to open the Digging Pa generator in a sub-window.

  4. Enter your search filters, then enter the amount of advances you are willing to search through into the + field, usually you will find a result that produces a 500,000 in less than 100,000 advances.

  5. Set the correct weather condition if you have not already done so. This should be set inside the Watt Trader sub window automatically if you followed step 2.

  6. Select the Consider Menu Close? checkbox, but do not calibrate your NPC count yet as we require a specific in-game setup (explained below).

Figure 2 - Game Setup

Getting Started (Game Setup)
  1. Head to the location pictured above, then interact with the Digging Pa NPC at least once to unlock to the fly point and get through their minigame introduction.

  2. Check your weather condition by opening the map. If the current weather is Rain or Thunderstorm be sure to change it as these conditions cause instability.

  3. Now you will need to position yourself optimally, this is done by using a movement technique that disrupts the NPCs "active" state and can achieved by doing the following:

    • Run towards the NPC and interact with them in one quick, smooth motion, then back out of the NPC dialogue.
    • If done correctly, you should see the advances occuring at a slow rate after backing out of the dialogue.
    • If done incorrectly, the NPC will activate after backing out of the dialogue, causing the usual, fast, NPC activation advances.

  4. Calibrate your NPC count as normal, the NPC count when standing next Digging Pa is variable and will change every day. Enter your calibrated NPC count into the Digging Pa sub window on owoow.

  5. Perform a date advance to refresh the daily highlight item, this can be done using owoow's date skipping features or the PvP date skipping exploit.

  6. Save the game. This will preserve the current NPC state upon resetting.

Advancing to the Target

In this example, a frame that produces x Watts can be seen in a {% include tooltips/clusters.html %} between advances x-y, these are the advances that we will aim to close the pause menu on and then interact with the NPC to lock in our desired result.

Step-by-Step
  1. Advance toward your target using any of the methods listed on this page.

    • Consider using wet weather or date skipping to advance large distances (for targets > 50,000 advances, it is unlikely you will need to use these methods).

    • Consider using NPC activation or menu close prediction to advance shorter distances (for targets < 50,000 advances, most targets will be within this range).

  2. Stop advancing once you are ~1000 advances before your target, then reposition yourself optimally as shown in Figure 1 if you have moved.

  3. Ensure that the Consider Menu Close? checkbox is checked in the Digging Pa sub-window, and confirm that your NPC count is calibrated correctly.

  4. Click Update Seeds on the main window of owoow, then re-open the Digging Pa sub-window and search for your target again.

  5. From this point, use animations to reach the exact target frame, then return to the main pause menu.

  6. Unpause the game and interact with the Watt Trader NPC to lock in your desired reward. Play the minigame and spam the A button to collect your Watts.
