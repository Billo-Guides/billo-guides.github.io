Introduction

Item RNG manipulation offers an efficient method for farming rare items and in-game currency. By accurately predicting potential outcomes, players can maximize the effectiveness of their farming efforts, ensuring optimal use of time when farming for rare in-game resources by removing the element of luck.

RNG Type Description Requirements
Cram-o-matic Farm rare PokeBalls (Apricorn, Safari, Sport) in the Isle of Armor Dojo.
  • Access to the Isle of Armor DLC
  • Access to the Cram-o-matic machine
  • Plenty of Apricorns/Items to use
Loto-ID Farm Master Balls, PP Max, and Rare Candies at any PokeCenter using Trainer Information.
  • Access to Wedgehurst PokeCenter
  • (Recommended) Pokemon from various trainers

Watt Trader Farm rare PokeBalls (Dream, Beast) at Snowslide Slope in Crown Tundra.

  • Have plenty of Watts (200,000+)
  • Have spoken to the Watt Trader NPC

  • Access to the Isle of Armor DLC
  • Have plenty Armorite Ore to spend
  • Have spoken to the Digging PA once

Setup Requirements

This section provides the required setup & installation instructions, read through this information carefully.
You Will Need:
Tools & Applications:
  • .NET 9.0
  • Sword/Shield Overworld RNG Tool (owoow)
  • ldn-mitm
Installation Instructions:
  1. Download .NET 9.0 and run the setup wizard until complete. This is a required program dependency.
  2. Download and extract owoow.zip into an fresh, empty folder, then extract ldn-mitm.zip into the same folder.
  3. Move the atmosphere and switch folders to the root of your SD card, agree to overwrite folders if prompted.
  4. Reboot the console into your chosen CFW environment (it is recommended to use SysCFW).
Program Connectivity Instructions:
  1. Launch owoow.exe, ensure that the program launches properly with no issues.
  2. Type the local IP address of the console into the Switch IP: field.
  3. Enter the overworld in-game, then click Connect on owoow after doing so.

RNG Tool Overview

This section provides program information and usage tips, read the information here if you are new.
Figure 2 - Owoow Overview

RNG Tool Overview:

Upon launching and connecting to owoow for the first time, you will see a navigation bar at the top of the page with multiple options. Each tool is clearly labeled, making it easy to understand its function.

Feature Overview:
  • Encounter Lookup - Only used for overworld manipulation; can be ignored.
  • Loto-ID - Used for manipulating the Rotom Lottery.
  • Cram-o-matic - Used for manipulating the Cram-o-matic.
  • Watt Trader - Used for manipulating the Watt Trader.
  • Digging PA - Used for manipulating the Digging PA.
  • Wailord Respawn - Used for respawning the Isle of Armor Wailord.

Although Wailord Respawn is not directly related to item RNG manipulation, however, it will be included in this guide as the underlying concepts are the same.
