Menu Close Prediction (CFW)

This page contains information about calibrating your NPC count and using that knowledge to accurately hit target frames. As this technique is frequently used throughout the guide, it is important to develop a solid understanding of how it works.

Closing and reopening the pause menu is a reliable technique used to advance the RNG state in a controlled and predictable manner. Typically, closing the menu also "freezes" natural RNG advancements for 2â€“3 seconds, which makes this method a highly consistent approach for targeting specific frames.

Figure 1 - Menu Close Demonstration

In Figure 1, we can observe that 25 advances were consumed from closing the pause menu (note, this figure will vary depending on the current RNG state, NPC count, and location). We now need to keep this figure in mind, and open the Menu Close Advances Viewer sub-window, located at the top of the main window.

Figure 2 - Menu Close Advances Viewer

In Figure 2, the sub-window has been opened. After searching, a table has appeared. The goal is to tweak the value in the NPCs field until we see that the first result inside the Jump column matches the figure observed earlier (in this case, we are tweaking the NPC field until we see +25 in the Jump column of the table).

Figure 3 - Finding NPC Count

After tweaking the value inside the NPCs field we can see that a +25 jump occurs when searching with 17 NPCs. You should now close and re-open the pause menu few times extra to make sure all of the values inside the Jump column align properly (see Figure 3).

Great! You should now have a basic understanding of how to calibrate NPC counts, and how to apply that knowledge to accurately hit target frames (you can leave this page open for future reference).
