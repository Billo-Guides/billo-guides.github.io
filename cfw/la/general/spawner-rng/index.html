Spawner RNG Legendary Abuse

This page will provide in-depth information about how to RNG abuse Legendary Pokemon in Legends: Arceus that are tied to what is known as Spawners. We will be using RNG generator scripts and tools written by Lusamine, as mentioned on the main page.

From here it is assumed that you have already followed all of the setup instructions found on the introductory page of this guide.

Script Setup:
For Spawner RNG you will need:
  • Spawner RNG calculation script
  • Spawner seed solver script
Spawner calculation script setup instructions:
  • Open LINQPad 7, then copy/paste the Spawner RNG calculation script into the main window.
  • Change the Language in the LINQPad header to C# program.
  • Press F4, then add PKHeX.Core.dll to the Additional References tab by selecting it via Add/Browse...
  • Type PKHeX.Core into the Namespace Imports tab.
  • Run the script by clicking the â–¶ button, You should now see the script output at the bottom of the window.

Seed reversal script setup instructions:
  • Open LINQPad 7, then copy/paste the Spawner seed solver script into the main window.
  • Change the Language in the LINQPad header to C# program.
  • Press F4, then add PKHeX.Core.dll, Microsoft.Z3.dll & libz3.dll to the Additional References tab by selecting it via Add/Browse...
  • Copy/Paste the following into the Namespace Imports tab:

    Microsoft.Z3
    PKHeX.Core
    System.Collections.Concurrent
    System.Runtime.CompilerServices
    System.Threading.Tasks
