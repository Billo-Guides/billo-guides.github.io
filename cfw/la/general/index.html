Legends: Arceus RNG

This guide provides in-depth coverage of how to RNG abuse Legendary Pokemon in Legends: Arceus, using RNG generator scripts written by Lusamine. It is possible to RNG manipulate Legendary Pokemon for desirable IVs & Natures, the method used can vary depending on if the encounter uses Main RNG or is tied to a Spawner.

From here it is assumed you are using a CFW Switch console running Atmosphere, and have sys-botbase correctly installed. This guide is also focused towards users running Windows OS.

Setup Instructions:
PC Setup
  • Moarencounterbots - Sysbot.NET fork by Lusamine
  • PKHeX & Plugins
  • PokeViewer.NET
  • LINQPad 7
  1. Extract the contents of Moarencounterbots.zip into an empty folder in a known location.
  2. Grab almsetup_stable.exe, move it into an empty folder, then run the file to begin downloading.
  3. Grab the latest release of Pokeviewer.NET, then extract the contents into a fresh, empty folder.
  4. Download LINQPad 7 and place it into an empty folder, this is used to run the RNG abuse scripts.
Script Setup:
  • PKHeX Development Build - For PKHeX.Core.dll
  • MicrosoftZ3.dll & libz3.dll - For other important .dll files
  1. Create a folder called dlls in a known location on your PC.
  2. Download PKHeX Development Build, then extract PKHeX.Core.dll into the folder.
  3. Download MicrosoftZ3.dll & libz3.dll then extract those into the same folder.
  4. You should now have all 3 of the relevant .dll files inside the same folder for easy access later on.
Switch Setup:
  • sys-botbase
  1. Download the latest release of sys-botbase and extract the atmosphere folder to a known location.
  2. Place the atmosphere folder onto the root of your Switch SD card, then reboot the console.
  3. You should now have sys-botbase properly installed! This is necessary for your switch to connect to your computer remotely.

Required Reading

There are 2 different ways that the game uses to generate Legendary Pokemon, these are Main RNG seeding, and Spawner RNG seeding. It is important to understand the difference between these two generation methods, and understand which encounters fall into each category before getting started.

Main RNG VS Spawners:
What is Main RNG?

Main RNG is a constantly ticking hidden state and so hitting the desired result will be dependent on correct/precise timing. Moarencounterbots can be used to accurately read and track your current Main RNG seeds in real time.

The following legendary encounters are tied to Main RNG:
  • Dialga or Palkia - Origin
  • Dialga or Palkia - Non-Origin
  • Giratina
  • Regigigas
  • Arceus
What is a Spawner?

Spawners hold a specific Pokemon that is visible in the overworld, when a Pokemon is knocked out the spawner seed will advance by 1 frame. This means that hitting a desired result will be based on KOing the Pokemon on the spawner a certain amount of times in a row until you reach your target frame.

The following legendary encounters are tied to Spawners:
  • Lake Trio - Uxie/Mesprit/Azelf
  • Weather Quartet - Tornadus/Thundurus/Landorus/Enamorus
  • Heatran
  • Cresselia
  • Manaphy/Phione
  • Shaymin
  • Darkrai
