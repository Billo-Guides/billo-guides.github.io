Dialga/Palkia RNG Abuse

This section of the guide aims to provide a worked example for how to RNG abuse base form Dialga/Palkia.

If you have made it this far, you should already have the initial setup completed and have a general understanding of how to use Moarencounterbots and the RNG abuse scripts.

Don't understand how to use the tools/scripts yet? Click here to go back!
Getting Started:

First, you will need to find a suitable seed and target frame. To do this will need Moarencounterbots and the Main RNG abuse script open and ready to use, including setting the appropriate IV/Nature filters.

  1. Open the Main RNG generator script, scroll down and input the correct Nature/IV filters.
  2. Stand outside the Origin Legendary encounter and save the game. You should be as close as possible to the encounter cutscene.
  3. Press the Home button to freeze the RNG state, and select EncBotCopySeedLA on Moarencounterbots.
  4. Add the bot, then click Start All to get your current seed, this will be automatically copied to clipboard.
  5. Paste the seed into the script, then set the amount of frames you wish to search through.
  6. Run the script, if you do not find a desirable result, you will need to reset the game and run EncBotCopySeedLA again.

In this example, we will be going for any Palkia-Origin with 6 perfect IVs (31/31/31/31/31/31), you can be more specific with your filters if you wish, such as targeting a specific Nature.
