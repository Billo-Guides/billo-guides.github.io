Main RNG Legendary Abuse

This page will provide in-depth information about how to RNG abuse Legendary Pokemon in Legends: Arceus that use Main RNG to generate internal values such as IVs and Nature. We will be using RNG generator scripts and tools written by Lusamine.

How To Use - RNG Script

By pressing the ▶ button to run the script, we are able to see a various different spreads, these spreads are for the seed used in the script example (E1CEA3FF1E3A9B5CB709ACB78FF71808). We can tell that the script is working as intended if results are displayed at the bottom of the LINQPad window.

To find your current Main RNG seed you will need to use Moarencounterbots. This program is vital for viewing and tracking your current seed in real time. Follow the step-by-step guide below to understand how to use the program to find your current seed.

Finding & Monitoring Your Current Seed:
Finding Your Current Seed:
  • Open Sysbot.NET and click the Hub tab, then open the EncounterLA dropdown menu.
  • Change DisplaySeedMode to Bit128.
  • Head back to the Bots tab and select EncBotCopySeedLA in the dropdown menu.
  • Type in your Switch local IP address, then add the bot (port should be 6000).
  • Open Legends: Arceus and press the Home button to freeze the RNG state.
  • Start the bot. You should now see your seed in the Logs tab, it will also automatically be copied to your clipboard!
Monitoring Your Current Seed:
  • Open Sysbot.NET and click the Hub tab, then open the EncounterLA dropdown menu.
  • Change the value inside MonitorRefreshRate to 0.
  • Head back to the Bots tab and select EncBotRNGMonitorLA in the dropdown menu.
  • Type in your Switch local IP address, then add the bot (port should be 6000).
  • Open Legends: Arceus and enter the overworld.
  • Start the bot. You should now see your seed is being monitored inside the Logs tab in real time!

Once you have all everything configured properly inside the Hub, you can switch between EncBotCopySeed and EncBotRNGMonitor as often as you like without needing any further configuration.

Now that we have our current seed, we need to input it into the RNG script and set a range of frames to search through.

Figure 2 - Main RNG Script Configuration



If you plan to search for specific IVs and Nature you will need to edit the filters, filters are located towards the bottom of the script, by default the filters are written as comments, so any search will be done without any filters applied.

A line of code that begins with // is a comment. To activate the filters, simply remove the // from the lines you wish to activate, to deactivate a filter, add // back onto the line.

You will also need to change the Nature and IV values if you wish to filter for specific combinations. We can safely ignore the gender filter since none of these legendaries have a gender.

Figure 3 - IV/Nature filtering



In this example we have applied filters for any 31/0/31/31/31/31 spread with a Calm nature by removing the comment tags on the IV and Nature filters. This seed has a lengthy amount of advances before hitting the target frame, so it can be considered undesirable. To refresh your seed you will need to restart the game and run EncBotCopySeedLA again until you find something desirable.
