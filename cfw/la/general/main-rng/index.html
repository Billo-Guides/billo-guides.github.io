Main RNG Legendary Abuse

This page will provide in-depth information about how to RNG abuse Legendary Pokemon in Legends: Arceus that use Main RNG to generate internal values such as IVs and Nature. We will be using RNG generator scripts and tools written by Lusamine.

From here it is assumed that you have already followed all of the setup instructions found on the introductory page of this guide.

Script Setup Instructions:
  1. Open LINQPad 7, then copy/paste the Main RNG calculation script into the main window.
  2. Change the Language in the LINQPad header to C# program.
  3. Press F4, then add PKHeX.Core.dll to the Additional References tab by selecting it via Add/Browse...
  4. Type PKHeX.Core into the Namespace Imports tab.
  5. Run the script by clicking the ▶ button, You should now see the script output at the bottom of the window.

How To Use - RNG Script

This section intends to help you understand how to use Moarencounterbots and the Main RNG generator script. You do not need any prior programming knowledge in order to get started.

As we can see, by pressing the ▶ button we are able to see a various different IV/Nature spreads, these are the upcoming spreads based on the seed used in the script example (E1CEA3FF1E3A9B5CB709ACB78FF71808).

Finding your current seed:

To find your current Main RNG seed/state, you will need to use Moarencounterbots. Moarencounterbots has the ability to read and track your current Main RNG state.

Step-by-step (finding your current seed):
  • Open the Sysbot.NET window and head over to Hub tab, then open the EncounterLA dropdown menu.
  • Change DisplaySeedMode to Bit128.
  • It is also advised to change the MonitorRefreshRate to 0 while we are in this menu.
  • Head back to the Bots tab and select EncBotCopySeedLA in the dropdown menu, type in your Switch local IP address, then start the bot.
  • Open Legends: Arceus and press the Home button on your controller to freeze the RNG state.
  • Start the bot. You should now see your seed in the Logs tab, it will also automatically be copied to your clipboard!

By default, DisplaySeedMode will be set to Bit32, which is not what the script expects so will be incorrect for this use case. By changing the MonitorRefreshRate to 0 milliseconds, we will be able to monitor our RNG state properly in real-time.

Troubleshooting

If you are running into issues with the bot connecting to your console, double check that your IP address is correct by opening the Homebrew menu on your console and pressing Y (you do not need to change the port, this should always be 6000).

If you continue to run into problems, consider visiting the Troubleshooting page of the Sysbot.NET Wiki.

If you plan to search for specific IVs and Nature you will need to edit the filters, filters are located towards the bottom of the script. By default the filters are "commented out" with //, we can tell that a line is commented out because it will be written in greentext.

To activate a filter, we will need to remove the comment. You will also need to change the Nature and IV values if you wish to filter for specific Nature/IV combinations. We can safely ignore the gender filter since none of these legendaries have a gender.
