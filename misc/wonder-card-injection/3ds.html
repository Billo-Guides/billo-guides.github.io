3DS Setup Instructions

Required tools & setup instructions for 3DS games (XY/ORAS/SM/USUM).
Tool Requirements & Installation
Necessary Tools
  • PKHeX
  • RunAsDate
  • ProjectPokemon Event Gallery Wonder Cards
  • A method of extracting your save file (make multiple backups!)
Setup & Installation
  1. Download PKHeX and extract it to a known location on your computer, launch PKHeX.exe and confirm that it opens properly with no issues.
  2. Download RunAsDate and extract it to a known location on your computer, launch RunAsDate.exe and confirm that it opens properly with no issues.
  3. Close PKHeX, you will now need to launch RunAsDate and select PKHeX as your program to launch.
    • Enter a time and date that matches your desired event distribution, this is an important step for legality purposes.
    • You can find information about the distribution period of an event on ProjectPokemon or Bulbapedia.
  4. Extract your 3DS save file and place it inside a known location on your computer, then open the save file using PKHeX (File -> Open -> Select the .sav file).
  5. Open the Event Gallery repository and open the folder that corrisponds to your game generation.
    • If you are injecting Wonder Cards into a Generation 6 game (XY/ORAS), open the Released/Gen 6/Wondercards directory.
    • If you are injecting Wonder Cards into a Generation 7 game (SM/USUM), open the Released/Gen 7/3DS/Wondercards directory.
  6. Select and download your desired Wonder Card. Check if your Wonder Card is Local Wireless/Infrared or serial Code/Nintendo Zone by researching the event on the ProjectPokemon Website.
    • If your Wonder Card was distributed locally or via infared, this generally means it can be injected into any save file regardless of region.
    • If your Wonder Card was distributed via serial code or Nintendo Zone, this generally means that the event was region locked.
    • For region locked events, ensure the console you'll be redeeming/performing the RNG on matches a legal Region for that given event Pokémon. If doing this on Emulator, you can manually change & set the Region for one that is legal in the settings before doing your RNG.

Wonder Card Injection

At this stage it is assumed that you have launched PKHeX using RunAsDate on a date that matches the official distribution period and that your save file is opened and ready to be edited. It is also assumed that you have done your due diligence to ensure that the Wonder Card you plan to inject will be legal on the save file you are injecting into.

Step-by-step instructions
  1. Open the save file in PKHeX if you have not already done so, then navigate to the SAV tab above the box viewer.
  2. Click the Mystery Gift button to open the Mystery Gift injection subwindow. This is where you will perform the injection.
  3. Click the Import button, then select your chosen Wonder Card in the file explorer, you should now see your Wonder Card in the subwindow.
  4. Right-click an empty Wonder Card slot in the PGT row then click Set to apply it to your save file. You can now press the Save button to close the subwindow.
  5. Export your save file (File -> Export Sav) and overwrite your existing save file, then load it back onto your cartridge/emulator.
  6. Visit the delivery man in-game. He can be found standing next to the desk/PC inside any Pokémon Center.

You can optionally save before collecting your gift if you wish to soft reset or RNG manipulate it for desirable values (IVs, Nature, etc).

Important - Important - In special cases, it is important to follow the proper in-game procedures rather than using shortcuts. For example, if you are injecting an egg Wonder Card and want to hatch it shiny on another save file, you should trade the egg between save files. Do not inject the .pk* file directly into the other save file, as this bypasses the intended game mechanics and may result in an illegal Pokémon.
