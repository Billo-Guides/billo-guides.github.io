DS Setup Instructions

Required tools & setup instructions for DS games (DPPt/HGSS/BW/B2W2).
Tool Requirements & Installation
Necessary Tools
  • PKHeX
  • BW_Tool (Entralink/Dream Radar)
  • RunAsDate
  • ProjectPokemon Event Gallary Wonder Cards
  • A method of extracting your save file (make multiple backups!)
Setup & Installation
  1. Download PKHeX and extract it to a known location on your computer, launch PKHeX.exe and confirm that it opens properly with no issues.
  2. Download BW_Tool and extract it to a known location on your computer, launch BW_Tool.exe and confirm that it opens properly with no issues.
  3. Download RunAsDate and extract it to a known location on your computer, launch RunAsDate.exe and confirm that it opens properly with no issues.
  4. Close PKHeX, you will now need to launch RunAsDate and select PKHeX as your program to launch.
    • Enter a time and date that matches your desired event distribution, this is an important step for legality purposes.
    • You can find information about the distribution period of an event on ProjectPokemon or Bulbapedia.
  5. Extract your DS save file and place it inside a known location on your computer, then open the save file using PKHeX (File -> Open -> Select the .sav file).
  6. Open the Event Gallary repository and open the folder that corrisponds to your game generation.
    • If you are injecting Wonder Cards into a Generation 4 game (DPPt/HGSS), open the Released/Gen 4/Wondercards directory.
    • If you are injecting Wonder Cards into a Generation 5 game (BW/B2W2), open the Released/Gen 5/Wondercards Fulls directory.
    • For Generation 4 games, select the folder that corrisponds to your game language, e.g select ENG if your cartridge/ROM is English.
    • For Generation 5 games you will need to look at the language tag on the .wc5full filename to confirm a match.
  7. Select and download your desired Wonder Card. For legality purposes a Wonder Card must be injected into a save file of the same language as the Wonder Card! Double check your download before continuing.

Wonder Card Injection

At this stage it is assumed that you have launched PKHeX using RunAsDate on a date that matches the official distribution period and that your save file is opened and ready to be edited. It is also assumed that you have done your due diligence to ensure that the Wonder Card you plan to inject matches your save file language.

Step-by-step instructions
  1. Open the save file in PKHeX if you have not already done so, then navigate to the SAV tab above the box viewer.
  2. Click the Mystery Gift button to open the Mystery Gift injection subwindow. This is where you will perform the injection.
  3. Click the Import button, then select your chosen Wonder Card in the file explorer, you should now see your Wonder Card in the subwindow.
  4. Right-click an empty Wonder Card slot in the PGT row then click Set to apply it to your save file. You can now press the Save button to close the subwindow.
  5. Export your save file (File -> Export Sav) and overwrite your existing save file, then load it back onto your cartridge/emulator.
  6. Visit the delivery man in-game, in Gen 4 titles he can be found inside the PokéMart, in Gen 5 titles he can be found inside the PokéCenter.

Figure 2 - In-game Redemption

You can optionally save before collecting your gift if you wish to soft reset or RNG manipulate it for desirable values (IVs, Nature, etc).

Important - Important - In special cases, it is important to follow the proper in-game procedures rather than using shortcuts. For example, if you are injecting an egg Wonder Card and want to hatch it shiny on another save file, you should trade the egg between save files. Do not inject the .pk* file directly into the other save file, as this bypasses the intended game mechanics and may result in an illegal Pokémon.
