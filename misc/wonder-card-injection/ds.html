DS Setup Instructions

Required tools & setup instructions for DS games (DPPt/HGSS/BW/B2W2).
Tool Requirements & Installation
Necessary Tools
  • PKHeX
  • RunAsDate
  • ProjectPokemon Event Gallary Wonder Cards
  • A method of extracting your save file (make multiple backups!)
Setup & Installation
  1. Download PKHeX and extract it to a known location on your computer, launch PKHeX.exe and confirm that it opens properly with no issues.
  2. Download RunAsDate and extract it to a known location on your computer, launch RunAsDate.exe and confirm that it opens properly with no issues.
  3. Close PKHeX, you will now need to launch RunAsDate and select PKHeX as your program to launch.
    • Enter a time and date that matches your desired event distribution, this is an important step for legality purposes.
    • You can find information about the distribution period of an event on the ProjectPokemon Website (use the search bar).
  4. Extract your DS save file and place it inside a known location on your computer, then open the save file using PKHeX (File -> Open -> Select the .sav file).
  5. Open the Event Gallary repository and open the folder that corrisponds to your game generation.
    • If you are injecting Wonder Cards into a Generation 4 game (DPPt/HGSS), open the Released/Gen 4/Wondercards directory.
    • If you are injecting Wonder Cards into a Generation 5 game (BW/B2W2), open the Released/Gen 5/Wondercards Fulls directory.
    • For Generation 4 games, select the folder that corrisponds to your game language, e.g select ENG if your cartridge/ROM is English.
    • For Generation 5 games you will need to look at the language tag on the .wc5full filename to confirm a match.
  6. Select and download your desired Wonder Card. For legality purposes a Wonder Card must be injected into a save file of the same language as the Wonder Card! Double check your download before continuing.

Wonder Card Injection

At this stage it is assumed that you have launched PKHeX using RunAsDate on a date that matches the official distribution period and that your save file is opened and ready to be edited. It is also assumed that you have done your due diligence to ensure that the Wonder Card you plan to inject matches your save file language.

Step-by-step instructions
  1. Open the save file in PKHeX if you have not already done so, then navigate to the SAV tab above the box viewer.
  2. Click on an unused Wonder Card slot then import the Wonder Card you wish to inject by selecting it in the file explorer.
  3. Right-click an empty WonderCard slot in the PGT row then click SAV to apply the Wonder Card to your save file, you can now close the subwindow.
  4. Export your save file (File -> Export Sav) and overwrite your existing save file, then load it back onto your cartridge/emulator.
  5. Visit the delivery man in-game, in Gen 4 titles he can be found inside the PokéMart, in Gen 5 titles he can be found inside the PokéCenter.
