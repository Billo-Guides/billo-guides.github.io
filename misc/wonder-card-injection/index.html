Safe Wonder Card Injection

This page covers how to inject WonderCards legally and safely in Generation 3-7 titles. A WonderCard is essentially a key to redeeming a specific event Pokemon, these "keys" are used to redeem a Pokemon from the Mystery Gift NPC found inside the PokeCenter or PokeMart.

Setup Instructions:
Required Tools:
  • PKHeX by Kaphotics
  • ProjectPokemon Event Gallery
  • ProjectPokemon Gen 3 WCTool
  • A copy of the save file you wish to inject into.

Download and extract the executable (.exe) file of both PKHeX & Gen 3 WCTool into an empty folder then run the application you plan to use (see below)

Extracting Your Save File:
  • GBA Guide
  • NDS Guide
  • 3DS Guide

If you are using an emulator, you will need to either save in game and locate where the save file data is stored, or export the save using the export function of the emulator (export as sav when possible).


Getting Started

Before starting, understand that this guide is intended to be used for personal use/collection purposes only. If you plan to inject Wonder Cards to trade the Pokemon in a community, be sure to understand the community rules surrounding this topic. I will not be held responsible any punishments that occur as a result of trading or misinjecting Wonder Cards.

Step-By-Step Intructions (GBA):
  1. Back up the save file multiple times using one of the above mentioned guides & methods.

  2. Find your desired WonderCard using the ProjectPokemon Event Gallery Repository, then download the file to a known location on your computer.

  3. Verify the details of the event using the search bar on ProjectPokemon, the event must match what is legally achievable (e.g Old Sea Map Mew is exclusive to Japan).

  4. Open the Gen 3 WCTool, then click Official Event Distributor, then open your extracted save file.

  5. If the region is incorrect when loading the save file, be sure to correct it now (see Old Sea Map Mew example)!

  6. Select the event to inject and click Inject and Save, then export the edited save and overwrite one of your existing save copies.

  7. Re-inject your save file onto your cartridge using your save manager/flashcart. You should now be able to complete the event as intended.

Tips - You can also toggle the event flags needed using PKHeX, if you feel comfortable doing so, using the Gen 3 WCTool is considered to be a more foolproof solution as it automatically marks events that can be legally injected into your console/region.


Step-By-Step Intructions (DS | 3DS):
  1. Back up the save file multiple times using one of the above mentioned guides & methods

  2. Find your desired WonderCard using the ProjectPokemon Event Gallery Repository, then download the file to a known location on your computer.

  3. Verify the details of the event using the search bar on ProjectPokemon, the event must match what is legally achievable (e.g Old Sea Map Mew is exclusive to Japan).

  4. Open PKHeX and drag your save file into the main window, alternatively you can use File -> Open SAV to manually open your save file. The contents of your save file should open at this point, if done correctly.

  5. Click the Sav tab located above the box viewer, then locate the Mystery Gift option, this should open a sub-window for Wonder Card injection.

  6. Click on an unused Wonder Card slot then import the Wonder Card you wish to inject by selecting it in file explorer and clicking Open, you should now see the Wonder Card you have chosen inside the sub-window.

  7. Right-click an empty WonderCard slot in the PGT row then click Set to apply the Wonder Card to your save file, you can now close the sub-window.

  8. Export your save file by clicking File -> Export SAV -> Export Main, overwrite one of your existing save copies.

  9. Re-inject your save file onto your cartridge using your save manager/flashcart, you should now be able to redeem the event as intended by visiting the Mystery Gift NPC inside the PokeCenter or PokeMart.

Note - In Gen 4 & 5, the language of the cart dictates what events can be legitimately injected (meaning an english distribution event can only be injected into an english game and so on).

Note - In Gen 6 & 7, Local Wireless/Infrared Events are always region free, while serial Code/Nintendo Zone events are usually region locked.
