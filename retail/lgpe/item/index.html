Hidden Item RNG (LGPE)

This guide provides coverage on how to RNG manipulate valuable items in Let's Go Pikachu & Eevee's Cerulean Cave and Celadon City Game Corner. This most useful application of this type of RNG is for farming Master Balls (Cerulean Cave), PP Max & Bottle Caps (Game Corner). The LGPE Item RNG Tool used throughout this guide was created by Lincoln-LM, this tool was created based on initial research from e52301147, Anubis & Lincoln.

From here you are using an unmodified Switch console running stock firmware.

Tool Requirements:
Tools:
  • LGPE Item RNG Tool
Instructions:
  1. Download and extract the RNG tool to a known location on your computer.
  2. Double click to the .exe file to run the program as normal.
It may take a few seconds for the program to open properly, if done correctly, the program will look similar to the images below.

Getting Started

After following the initial setup instructions, you will need to meet the in-game requirements found in the box below.

Figure 1 - In-Game Setup:



In-Game Setup
You Will Need:
  1. Access to Cerulean Cave or the Celadon City Game Corner.
    • Cerulean Cave for Master Balls, Game Corner for Bottle Caps and PP Maxes.

  2. TM 05 (Rest) Taught to Partner Pikachu or Eevee.
    • Used for seed identification, teach rest to Pikachu or Eevee then put it to sleep by entering a battle and using Rest. Switch the Pokemon out and complete the battle after sleep status has been inflicted.

  3. Spare Repels to use in Cerulean Cave.
    • Wild Pokemon spawning produces unwanted and unpredictable advancements, if you plan to RNG manipulate Master Balls, ensure that you have plenty of repels.

  4. Items respawned in the area you wish to manipulate.
    • Items in Cerulean Cave have a chance to respawn every 256 steps, Game Corner items refresh on a new day. You can automatically refresh Game Corner items by changing the system date while searching for a local trade.

  5. Navigate to the correct location and save, then reset the game.
    • Seed identification is best done by starting from a freshly reset game on a brand new day, ensure the game is saved and reset after following all the other setup instructions. This step will need to be repeated every time a new day passes (00:00 on the system clock).

A list of table item rates and spots for item spawners can be found here, to check if an item has respawned, stand next to the item spawner and check to see if the tail of your partner Pokemon is waggling.

Figure 2 - Seed Identification



Seed Identification

The above image displays the LGPE Item RNG tool, to find your current seed, you will need to use the Seed Finding tab. Finding your seed can be done one of two ways:

  1. Observing "sleep messages".
    • With your partner Pokemon asleep, pause the game and select "Play with Pikachu" or "Play with Eevee" depending on the game you are playing, and observe sleep messages.

  2. Observing the candy obtained from transferring Pokemon.
    • Transfer 128 Venomoth one-by-one to Professor Oak and observe the candies given. Venomoth is currently the only Pokemon supported by the RNG Tool, due to being the most common Pokemon that works for seed identification.
Seed Identification - Sleep Messages:

This method is recommended as the requirements are a lot less strict and time consuming, the only requirement is that you have saved and reset the game with your partner Pokemon asleep. The example image above uses this method

  1. Open the game and enter the overworld, if you are planning to manipulate items in Cerulean Cave, use a repel and save the game, then reset and enter the overworld again.
  2. Open the Seed Finding tab of the LGPE Item RNG Tool, then select Partner Observations Initial State from the dropdown menu.
  3. Pause the game then click "PLAY WITH PIKACHU / PLAY WITH EEVEE", then take note of the message produced.
  4. If the message states "sleeping peacefully", type p into the Partner Observations (d/p) field.
  5. If the message states "in a deep sleep", type d into the Partner Observations (d/p) field.
  6. Repeat this process 64 times until a seed has been produced. Keep the game paused after identifying your seed to avoid any unwanted advancements.
Be quick to exit the menu after observing the message, staying in this menu over 3-5 seconds will cause extra unwanted advancements, observe the message, quickly back out, then note the message with a p or a d in the tool.
Seed Identification - Candy Transfers:

This method is slower, but is a more reliable. It is guaranteed to find the correct seed if monitored correctly. Before starting, save with the required 128 Venomoths caught in case of any inaccurate observations.

  1. Open the game and enter the overworld, if you are planning to manipulate items in Cerulean Cave, use a repel and save the game, then reset and enter the overworld again.
  2. Open the Seed Finding tab of the LGPE Item RNG Tool, then select Candy Observations Arbitrary State from the dropdown menu.
  3. Open Pokemon Box, then find the group of Venomoth you wish to transfer.
  4. Transfer the Pokemon one-by-one, and observe the candies given by Professor Oak.
  5. If a Quick Candy was produced, type q into the Candy Observations (q/s) field.
  6. If a Smart Candy was produced, type s into the Candy Observations (q/s) field.
  7. Repeat this process 128 times until a seed has been produced. Keep the game paused after identifying your seed to avoid any unwanted advancements.
You can use Candy Observations Arbitrary State instead if you wish. This requires less Venomoth transfers, but can result in seed inaccuracies on rare occasions.

Figure 3 - Seed Found



Searching For Targets

After a seed has been produced, open the Item Prediction tab, then select the item you wish to manipulate from the dropdown menu, and paste the seeds that were produced earlier into the assigned fields.

Set the advance range to a comfortable amount (1000 recommended), and the Max Partner Menu Opens to 1, then click the search button to check for results that meet the search criteria.

If no desirable result has been found, consider increasing the search range. If you do not want to increase the search range, restart the game and re-identify your seed again.

Figure 4 - Target Found



A target has been found, a Master Ball in x advances, you can now begin advancing the RNG state.

Advancing The RNG State

The following actions are considered to be the best methods to advance the RNG state for both consistency and speed:

Advancement Method Description Of Method
Follow Pokemon Interaction Using the Pokemon follow feature to either ride a Pokemon or have it trail behind you, then interacting with it.
Spawning Wild Pokemon Spawning wild Pokemon in the overworld will advance the RNG state a certain amount depending on the Pokemon spawned.
Pikachu/Eevee Interaction Opening the "Play with Pikachu/Eevee" menu will advance the RNG state in small increments, useful for the last few advances.
Figure 5 - Advancing RNG State



Advancing RNG State

Use any of the advancement methods in the table above to advance the RNG state. Seeds can be re-identified at any point by doing the following:

  1. Open the Seed Finder tab again and click the reidentification checkbox.
  2. Enter the last known RNG state into the Reidentification State fields.
  3. Observe "sleep messages" again, if you have already forgotten how to do this, the method can be found here.
  4. After a new RNG state is produced, enter it into the Item Prediction tab and search for your target again.
This process can be repeated as many times as needed, it is recommended to stop advancing the RNG state at random once you are within ~50 advances of the target.

Figure 6 - Close To Target



Final RNG State Advances

After re-identifying the current RNG state, the RNG tool displays that we are now x frames away from the target, and we have already relocated to a hidden item spot that can potentially contain the desired item.

We now need to advance the RNG state in a predcitable manner to reach the seed/advance that contains the desired item. As a reminder, this can be done in one of two ways:

  1. Viewing partner sleep messages (a consistent +2 advance).
  2. Transferring Venomoths (a consistent +1 advance).

Use sleep partner messages to get as close as you can to your target. If you are an even number of advances away from the target, you do not need to use Venomoth transfers. Venomoth transfers can be used to do the final advance if you are an odd number of advances away from the target.


Figure 7 - Result Found!



According to the search results, a Master Ball will be produced after interacting with the item spot on advance x. Our calculations indicate that y advancements have occured (from doing y partner message advancements), so it is now time to interact with the hidden item spot to claim our item.

A Master Ball was produced as expected, that's all there is to RNG manipulating valuable items in LGPE on retail hardware!
