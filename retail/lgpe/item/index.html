Hidden Item RNG (LGPE)

This guide provides coverage on how to RNG manipulate valuable items in Let's Go Pikachu & Eevee's Cerulean Cave and Celadon City Game Corner. This most useful application of this type of RNG is for farming Master Balls (Cerulean Cave), PP Max & Bottle Caps (Game Corner). The tools and scripts used throughout this guide were created by Lusamine.

From here it is assumed you are using a CFW Switch console running Atmosphere.

You Will Need:
Tools & Scripts:
  • LINQPad 8
  • LGPE Hidden Floor Item Script
  • PKHeX Development Build (For PKHeX.Core.dll)
  • LGPE Seed Solver
Installation instructions:
  • Download and install LINQPad 8, then create a desktop shortcut or find the location it has been installed to.
  • Open the LGPE Hidden Floor Item Script in your browser, specific setup instructions are covered below.
  • Download the LGPE seed solver, then extract cotents to a known location on your computer.
  • Download the PKHeX Development Build, then extract PKHeX.Core.dll into a known location on your computer.
PKHeX.Core.dll is a required dependency for the LGPE Hidden Floor Item LINQPad Script, it does not function properly without it.

In-game Requirements

It is recommended to complete this list of in-game requirements before starting the guide to save time later on:

  • Captured 128 Venomoth (used for seed identification).
  • Access to Cerulean Cave and/or Celadon Game Corner.
  • TM 05 (Rest) taught to partner Pikachu or Eevee.
  • Spare Repels to use in Cerulean Cave.
Why do I need to capture 128 Venomoth?

The method of finding your current seed involves transferring 128 Pokemon, no easier or less time consuming method has been found at this point in time.

Save the game after capturing the required 128 Pokemon so you can reset as a failsafe.

Ivysaur, Venusaur, Charmeleon, Wartortle, Butterfree, Beedrill, Ninetales, Parasect, Venomoth, Muk, Seadra & Tauros can also be used for seed identification, but Venomoth is recommended due to higher spawn rates.

Configurations & Setup

Agter completing the installation and game setup instructions, you will need to use LINQPad to configure the RNG scripts.

Figure 1 - Script Configuration



LINQPad Script Setup:

Upon launching LINQPad, you should see something similar to the above image. The links at the top of this page displays links for the scripts used in this guide, they can all be configured the same way.

Initial Script Setup
  • Change the Language (found at the top of the window) to C# program, delete any existing code that appears from doing so.
  • Copy the LGPE Hidden Floor Item Calculator script directly into the main window, then press F4 to open the Query Properties sub-window.
  • In the Additional References tab, click the Add/Browse button and select PKHeX.Core.dll (downloaded earlier).
  • Open the Namespace Imports tab then type PKHeX.Core.
  • Click the OK button at the bottom left of the sub-window to confirm these changes.
  • Run the script, you should see that an output has been produced at the bottom of the main window if setup was done correctly.
  • Repeat this process for all the scripts individually and confirm that they are all functioning as intended.
Ensure that the script is being ran as a C# Program, this is visible in the top bar of the main window. Use Figure 1 as a reference point if you are struggling with leveraging PKHeX.Core.dll.

After making sure the script works as intended, it is recommended to become familiar with how they work, such as modifying search filters and item tables in the Floor Item Prediction Script.

Figure 2 - Script Filters



Floor Item Prediction Script Filters:

By default the LGPE Hidden Floor Item script is configured to search for Master Balls within 2000 advances, using the Cerulean Cave Pokeball table. If you are seeking a different item these filters can be by doing the following:

  • To modify the table, change line 16 to match the table you wish to use, possible table selections are written directly above line 16.
  • To modify the selected item, change line 41 to match the item you wish to filter for, the entered name must be exact.
  • To adjust the amount of frames the script will search through, modify the number in line 20, it is not recommended to increase this number.

Examples of how to change each of the search filters can be found in Figure 4.


You should now have the script properly setup and have a general understanding of usage (such as applying search filters). Next, you will need to ensure that the item you plan to manipulate able to be picked up.

Figure 3 - Game Setup



Game Setup:

The item given from interacting with a hidden item spot is determined on the frame it was picked up. For this reason your item needs to be spawned and able to be picked up before doing anything else. In this example we will be searching for a Master Ball in Cerulean Cave.

Item Confirmation

Pikachu/Eevee will perform a tail wagging animation if the player is stood next to a hidden item, this animation is especially useful when respawning items in Cerulean Cave as the respawn is not guaranteed.

Stand next to your desired item spawner and confirm that an item is present by observing the tail of your partner Pokemon, once this has been confirmed, save the game.

  • In Cerulean Cave, spawners are assigned to drop specific items, and the pool will never change (a Pokeball spawner will always drop Pokeballs, for example).
  • In the Game Corner, the berries will vary between spawners but the rare items can be found using any of the 6 hidden item spawn spots.

Drop rates & Locations for Game Corner & Cerulean Cave item spawners can be found in this google sheet. If you have already collected the items, see below for item respawn mechanics.

Respawn Mechanics

The conditions to respawn collected items is different depending on location. For Cerulean Cave, items have a chance of spawning every time the player walks 256 steps. In the game corner, the items are respawned each time a day has passed.

To trick the game into believing that a day has passed naturally (which will respawn the items in the Game Corner immediately if yours are despawned), you will need to follow these steps:

  1. Open the pause menu and click Communicate.
  2. Select the Play with others option.
  3. Select Nearby Player, then Link Trade.
  4. Enter any link code, then search for a trade.
  5. Press the home button and navigate to System Settings.
  6. Change the system date forward by one day, this will respawn the items.

Great! At this stage everything has been setup properly, you can now begin the RNG manipulation process.

The Process

To manipulate a desired result, you will first need to understand how to calculate, identify and advance your current seed.

Seed Identification

To find your current seed, you will need to use the -filler- (pictured above) to record 128 actions in a row, doing so will produce a seed that can be pasted directly into the LINQPad script to search for results.

Step-by-step Instructions:
Keep the game paused, or identify your seed in an area where there are no wild Pokemon, such as a city/town/in-door location, to avoid unwanted advancements after identification.

Figure 5 - Searching For A Target



Searching For A Target:

To search for a target, identify your seed using the method mentioned above, then paste it into the assigned field on the LINQpad script, then click the run button (▶️ button). When the script has finished, results will be produced at the bottom.

Undesirable Result?
  • Reset the game, and enter the overworld again.
  • Identify your current seed again and search for results.
  • Repeat until a desirable result has been found.
It is recommended to search for a cluster. A cluster is multiple frames in a row that produce the same end result. This provides more opportunities to spawn the correct item.

Advancing Towards The Target

The following actions are considered to be the best methods to advance the RNG state, for consistent advancements and speedy advancements:

Advancement Method Description Of Method
Follow Pokemon Interaction Using the Pokemon follow feature to either ride a Pokemon or have it trail behind you, then interacting with it.
Spawning Wild Pokemon Spawning wild Pokemon in the overworld will advance the RNG state a certain amount depending on the Pokemon spawned.
Pikachu/Eevee Interaction Opening the "Play with Pikachu/Eevee" menu will advance the RNG state in small increments, useful for the last few advances.
