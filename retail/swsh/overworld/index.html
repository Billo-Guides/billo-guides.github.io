Overworld RNG Guide (RETAIL)

This guide provides in-depth coverage of how to use SWSH OWRNG Generator GUI by LegoFigure11 to RNG abuse wild encounters in Sword/Shield. With overworld RNG abuse, you can obtain Shiny or Marked Pokémon and calculate IVs, Nature, and Gender.

From here it is assumed you are using an unmodified console on stock firmware.

Running a modified console? Click here for the CFW focused guide!
Setup Instructions:
You will need:
  • SWSH Overworld RNG Generator
  • .NET 6.0
  • A calculator
Installation Instructions:
  1. Extract SWSH_OWRNG_Generator_GUI.zip into an empty folder on your PC in a known location.
  2. Download .NET 6.0 Runtime, then run the executable file and follow the setup instructions.
  3. Launch SWSH OWRNG Generator GUI and make sure it works as intended.

RNG Tool Setup

This section will cover the essential details you need to know in order to use the Generator. If you do not yet have a complete understanding of how each of the tabs, follow along with this section for relevant information & usage tips.

Figure 1 - Main Window Overview


Main Window Overview

The main window (pictured above), calculates upcoming target frames based on your search parameters. Here, we can assign encounter and search filters to search for specific target frames.

Feature descriptions:
  • The TID & SID fields expect your 5 digit TID/SID combination, you will need this to find Shiny Pokémon.
  • Init. adv. is where you enter the amount of advances that have passed since you began monitoring your seed.
  • The + field is the amount of frames to search through, you should set this to a comfortable amount.
  • Click the Shiny Charm checkbox if you have the Shiny Charm in your save file, otherwise leave unchecked.
  • Click the Mark Charm checkbox if you have the Mark Charm in your save file, otherwise leave unchecked.
  • The rest of the details on the left-hand side are encounter filters, these can be filled automatically using the Encounter Lookup sub window (explained below).
  • The middle section contains search filters, here you can set filters for specific Shininess, IVs, Nature, Marks, etc.
The Connect, Read Encounter & Skip features are exclusive to users running Custom Firmware (CFW), attempting to use these features on a retail device will cause issues.

Figure 2 - Retail Seed Finder Overview


Retail Seed Finder Overview

The Retail Seed Finder sub-window is where players can calculate their current seed, this is done by recording a sequence of 128 motions/animations in a row.

What are motions/animations?

This refers to an in-game feature on the Pokémon summary screen or party menu. Pressing L3 (pressing the left stick) will cause your Pokémon to perform one of 2 animations, these will be referred to as "physical" and "special" animations.

It is recommended to pick a Pokémon with a clear physical/special animation to record your seed with, good examples include Drilbur or Espeon. Animation differences can be found in the below image if needed (figure 3).

Seed finding instructions:
  1. Open the pause menu in-game, then open the Retail Seed Finder sub-window on SwSh OWRNG Generator GUI.
  2. Open the party menu, select the Pokémon you will use to record animations, then click Summary.
  3. Use the L3 button to cause an animation, take note of if the animation was physical or special.
  4. If the animation was physical, press the (0) Physical button. If the animation was special, press the (1) Special button.
  5. Repeat this process 128 times in a row and a seed will be produced. Click the Update Main Form button to paste it into the main window.
  6. If a seed was not produced this indicates that the animations were not recorded properly, try again.
Recording animations through the Pokémon summary screen is recommended, this is due to the party menu advancing +1 per button clicked, the summary screen advances +1 per animation, making it easier to keep track of.

Figure 4 - Encounter Lookup tab overview


Encounter Lookup Overview
This is the Encounter Lookup window. Searching for a specific species in the dropdown list will bring up a list of every possible wild encounter for that Pokémon in the entire game.

How to use the Encounter Lookup:
  1. Open the Encounter Lookup window and set the game version to the game you are currently playing.

  2. Type the species name of your chosen target Pokémon, then select it from the dropdown menu. You should now see a list of all possible encounters for that specific Pokémon in the table.

  3. Choose an encounter type; Static, Symbol or Hidden. Once you have decided, select one of the rows from the list by single clicking on it.

  4. The details of the encounter should now be automatically filled into the main window, and the Encounter Lookup window can now be closed.


Figure 5 - Menu Close Advances Viewer window overview


Menu Close Prediction Overview
The Menu Close Advances Viewer can be used to predict the amount of advancements that will pass from closing the pause menu. Closing the pause menu also has an added effect of freezing the RNG state in the area for a a few seconds, this makes it a great option for closing in on any target frames as it provides a lot more consistency in doing so.

By knowing how many advancements will occur from closing the pause menu, and having a window of silence to run into position and spawn the encounter, we can near guarantee to hit any potential target frames.

How to calculate the number of NPCs in an area:
  1. Attempt to count the number of NPCs in the area of your target manually if possible. If this is impossible, starting from 0 NPCs and working forward is acceptable.

  2. Identify your current seeds using the Retail Seed Finder, as explained above, keep the game paused after doing so. This will ensure that the amount of advancements that have occurred since finding your seed will remain at 0.

  3. Click the Menu Close Advances Viewer button at the top of the generator to open a smaller window.

  4. Take your seeds from the State[0] & State[1] fields of the Main Window, and enter them into the same fields of the Menu Close Advances Viewer window.

  5. Take the number of NPCs you counted earlier and put it into the NPCs field.

  6. Enter your number of advancements into the Init Adv. field. If the above steps were closely followed, this number should be 0.

  7. Enter the search range into the + field, then hit search, you should then close and reopen the menu and re-identify your current seed.

  8. Subtract the amount of advancements that were consumed by re-identifying your current seed from the total amount of advancements that have occurred. (For example, if 18 advancements have occurred, and it took 14 advancements to re-identify the current seed, then 4 frames passed from opening and closing the pause menu)

  9. If the values do not match, tweak the number in the NPCs field until it matches. (Decrease the value if the number of advances you got was lower than expected, increase if it was higher)


Advancing the RNG State

The RNG state will naturally advance, and the pace of the advancements will vary depending on your location, the weather of the location, and the spawned Pokemon that are present. In all cases, it's best to target a Pokémon in a "quiet" area with calm weather conditions. However, sometimes you will be forced into an unstable environment due to a lack of options.

Examples:
 - In Crown Tundra, the area is naturally quiet, so we will have a relatively easy time hunting a Pokémon in clear weather conditions.
- In Isle of Armor, the area is naturally noisy, so we will have a more difficult time hunting Pokémon in general.
- If we added rainy weather to either of these situations, we would increase the amount of natural noise and further increase the instability.

Day Skipping Information:
Day Skipping
Date skipping on retail is a manual process that requires some setup. In order to day skip efficently you must exit a ranked battle.

The amount of advancements from day skipping is heavily influenced by the amount of Pokémon currently stored within boxes. Fill your boxes with Pokémon to make the most of this feature. This does not include Pokémon in eggs!

Avoid date skipping in large distances inside any of the Wild Areas, including Isle of Armor and Crown Tundra Wild Areas. This may cause your game to crash due to the constantly advancing Max Raids! We recommend using an in-door location to advance distances >1000 days.

Day skipping will change the weather in the area! Be sure to set your date back to how it was originally to restore the weather condition.

How to perform the efficent day skipping:
  1. Pause the game and press VS to enter a ranked battle.

  2. Forfeit once you enter the battle. Alternatively you can play out the whole battle if you wish.

  3. From here navigate to the System Settings and change the date ahead by one day.

  4. Exit back to the main menu of the home screen and reopen the game.

  5. Repeat step 4 and 5.

  6. You should notice time-based events such as the Rotom Lottery function have reset, this is an indication that the exploit has been properly activated.
