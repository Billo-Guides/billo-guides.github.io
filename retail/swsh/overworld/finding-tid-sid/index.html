Sword/Shield Finding 5 Digit TID/SID

This tutorial provides a range of methods for how you can find your 5 digit TID/SID combination in Sword/Shield. Knowing these values is necessary if you wish to RNG abuse Shiny Pokémon from overworld encounters.

Possible Methods:

It is not possible to RNG abuse your TID or SID in Sword/Shield. This tutorial is only for finding your internal TID/SID on an existing save file. There are three possible methods you can use.

Using SysBot.NET

SysBot.NET is a bot service hosted by members of the community, typically found in Discord servers. If you do not have access to a SysBot, you can find a public server with a quick Google search.

SysBots come with dump functionality, which dumps the data of the shown Pokémon into a .pk8 file, this file is then shared into your Discord messages with the bot.

The file can then be manually inspected using a program like PKHeX to view your TID/SID combination. This method is recommended as it takes the least effort, but does require a Nintendo Switch Online subscription.

Help from a friend

If you have a friend with a CFW Switch console, they are capable of dumping the file for you, but they will need access to PKHeX to do so.

You can also have a friend check your TID/SID combination if you do not have access to PKHeX. PKHeX only works on Windows devices. This method is only recommended if you do not have any access to a Sword/Shield SysBot.

Finding a shiny Pokémon without RNG abuse

If you do not have access to the above options at all, you will need to manually search for a shiny Pokémon in the wild. It is possible to trace back to the frame that generated the shiny using SWSH OWRNG GENERATOR GUI.

This method is the most time consuming and luck dependent, and requires you to leave the game open immediately after capturing a shiny Pokémon.


Pick the method you wish to use, and find the relevant section of the page that applies to you down below.

Finding TID/SID using CFW.

As mentioned above, by having access to a CFW Switch console we can easily find our TID/SID combination.

As mentioned above, a Discord bot such as Sysbot.NET is capable of dumping Pokemon data for use with PKHeX, and from there we can easily find the TID/SID combination of the dumped Pokemon. You can find many servers that are hosting Sysbot.NET by a quick Google search.

Remember that bot up-time is not guaranteed! Each bot has a real Switch device dedicated to running it's services.

Alternatively, if you are not running a Windows machine, or do not have access to a Discord account, you can ask a friend running Switch CFW to use PKHeX to find your 5 digit TID/SID combination for you.

Requirements & Tutorial:
Requirements:
  • PKHeX
  • Nintendo Switch Online Subscription
  • A Discord account (If using Sysbot.NET)
Tutorial:
  1. Download and extract the PKHeX.exe into an empty folder and run the program.

  2. Click Options at the top of the window, then click Settings, change the Blank Save Version to Sword or Shield.

  3. Dump the data of any Pokémon that you have caught on your save file with the help of a friend or Sysbot.

  4. Download the file and open it in PKHeX (right click the Pokémon + click view if you dragged it into a box slot).

  5. Head over to the OT/Misc tab, you should see the trainer data of your Pokémon on-screen.

  6. Hover over the TID/SID fields with your mouse, your 5 digit TID/SID will be displayed.

Using A Shiny Pokémon

If you are unable to use a bot, or gain access to a console running Switch CFW, you will need to find a shiny Pokemon in the wild in order to find your TID/SID combination. This method should only really be used if you are unable to use the recommended methods.

Requirements & Tutorial:
Requirements:
  • Finding a shiny Pokémon in the overworld at random
  • Capturing the Pokémon, and leaving the game open (paused) after doing so
Tutorial:
  1. Run around in the overworld and encounter Pokémon until a shiny Pokémon appears.

  2. Capture the Pokémon, then quickly pause the game after doing so in order to pause the RNG state.

  3. Identify your current seed using the Retail Seed Finder window, then click the Search for TID/SID checkbox on the main window.

  4. Enter your search range, this should be however many advancements you believe has occurred since you encountered the Pokémon (If the above steps were closely followed, this should be a few thousand advancements at maximum).

  5. Hit search, and find the frame that generated your shiny Pokémon. You should now also see the TID/SID combination that made the Pokémon shiny.

  6. Take note of these values, you will need them in order to hunt shiny Pokémon, and to avoid repeating this process every time.

Once you have found your 5 digit TID/SID be sure to note it down, you can now begin to RNG abuse shiny overworld Pokémon!
