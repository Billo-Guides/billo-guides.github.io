5 Digit TID/SID

This tutorial provides a range of methods for how you can find your 5 digit TID/SID combination in Sword/Shield. Knowing these values is necessary if you wish to RNG abuse Shiny Pokémon from overworld encounters.

Possible Methods:

It is not possible to RNG abuse your TID or SID in Sword/Shield. This tutorial is only for finding your internal TID/SID on an existing save file. There are three possible methods you can use, these are:

Using SysBot.NET

SysBot.NET is a bot service hosted by members of the community, typically found in Discord servers. If you do not have access to a SysBot, you can find a public server with a quick Google search.

SysBots come with dump functionality, which dumps the data of the shown Pokémon into a .pk8 file, this file is then shared into your Discord messages with the bot.

The file can then be manually inspected using a program like PKHeX to view your TID/SID combination. This method is recommended as it takes the least effort, but does require a Nintendo Switch Online subscription.

Help from a friend

If you have a friend with a CFW Switch console, they are capable of dumping the file for you, but they will need access to PKHeX to do so.

You can also have a friend check your TID/SID combination if you do not have access to PKHeX. PKHeX only works on Windows devices. This method is only recommended if you do not have any access to a Sword/Shield SysBot.


Finding TID/SID using CFW

As mentioned above, by having access to a CFW Switch console we can easily find our TID/SID combination. The method will differ slightly depending on if you are using a SysBot, or asking a friend to help.

Keep in mind that SysBot hosts are not constantly available. Join a public server that contains an active Sword/Shield bot, then follow the instructions given by the bot host. If you do not have a Discord account you will need to make one or ask a friend for help instead.

Using CFW - Instructions
Requirements:
  • PKHeX
  • Nintendo Switch Online Subscription
  • A Discord account (If using SysBot.NET)
Tutorial:
  • Extract the contents of the PKHeX download into an empty folder in a known location on your PC.
  • Click Options at the top of the window, then click Settings, change the Blank Save Version to Sword or Shield.
  • Dump the .pk8 file of a Pokémon that was caught in the save file you wish to check the TID/SID on with a SysBot or with a friends help.
  • Download the file and drag it into the PKHeX window, you should now see the details on screen. Head over to the OT/Misc tab to view Trainer Information.
  • Hover over the TID field with your mouse, here you can see the 5 digit TID/SID combination. Take note of these values in a notepad file or similar.

Figure 1 - TID/SID Lookup

In this example we can see that the TID/SID combination is 28888/10267, now save this inside a text document, and enter the values into the assigned fields on Owoow.

Figure 2 - Program Entry

After entering your ID values into owoow, select your game version and Shiny/Mark Charm checkboxes if those items are present within your save file. Upon restarting the program these details will be saved for future use.
